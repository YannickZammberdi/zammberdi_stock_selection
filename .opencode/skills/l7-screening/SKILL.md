# 行业分析框架（L7-L10）

## 用途

输入**市场 + 一个行业**，输出一份完整的行业分析HTML报告，含五个Tab：

| Tab | 对应框架 | 内容 |
|-----|---------|------|
| 总结论 | — | 行业速览 + 综合评分 + 核心结论 |
| L7 量化初筛 | L7 | 六道筛子过滤，候选池 + 观察池 |
| L8 行业画像 | L8 | 产业链 + 市场规模 + 生命周期 |
| L9 竞争格局 | L9 | 市场结构 + 壁垒 + 驱动力拆解 |
| L10 周期时机 | L10 | 周期位置 + 估值 + 政策 + 买入清单 |

## 输入格式

用户给出行业名称和目标市场（未指定时默认为A股）。

```
"筛一下A股光伏"     → 市场=A股，行业=光伏
"ASX金矿看看"       → 市场=ASX，行业=金矿
"锂矿现在能买吗"   → 市场=A股，行业=锂矿
```

若行业太大（>200只），收紧阈值或优先用ASX 300/A股沪深300成分股作为筛选起点。

## 输出文件

完整行业分析报告存入 `stock-learning/stocks-analysis/行业报告/{市场}——{行业名}——行业分析报告.html`。

## 参考模板

参考 `stock-learning/stocks-analysis/行业报告/A股——光伏——行业分析报告.html` 的HTML结构和CSS样式。模板已包含完整的CSS、tab-bar和JS交互，直接复制后替换数据和内容即可。

## Tab结构

```
.report-wrap (max-width:800px)
  h1 + .subtitle + .meta
  .tab-bar (sticky top)
  .tab-content#tab-summary > .tab-inner
  .tab-content#tab-l7 > .tab-inner
  .tab-content#tab-l8 > .tab-inner
  .tab-content#tab-l9 > .tab-inner
  .tab-content#tab-l10 > .tab-inner
  .footer
```

CSS/JS与参考模板完全一致。meta中日期统一 `YYYY-MM-DD` 格式。

## 内容填充规则

### 总结论 tab

- **行业速览表：** 行业名、市场、L5分类、上市公司数+L7通过数、总市值、产业链利润核心环节、生命周期阶段、当前周期位置
- **综合评分表：** 6个维度各15-20分（行业规模与空间20/产业链位置15/竞争格局20/增长驱动力15/周期位置15/政策环境15），总分100分。每行含进度条（`bar-wrap`/`bar-bg`/`bar`），右侧写说明。末行总分加粗，颜色随分数变化（≥60绿，<60橙）
- **verdict：** 总分显示，≥80绿（优秀）、≥60橙（一般）、<60红（回避）
- **核心结论：** summary-box，写行业值不值得研究、用什么框架、当前能不能买
- **行业公式：** highlight-box，收入 = 量因子 × 价因子 + 3-5个关键跟踪指标

### L7 tab

- **筛子调整说明：** 6个筛子的标准值和调整原因。行业类型调整速查：

| 行业类型 | 跳过筛子 | 调整筛子 |
|---------|---------|---------|
| 银行/保险 | 毛利率, 负债率 | 估值用PB |
| ASX矿业 | 毛利率 | ROE放宽≥5%, 估值用PB |
| 地产 | — | 负债率放宽≤75% |
| 周期股底部（亏损） | — | 估值用PB替代PE |
| 亏损但高营收成长 | ROE | 估值用PS≤10x |

- **六道筛子标准：**

| 筛子 | A股主板 | A股创业板/科创板 | ASX |
|------|---------|----------------|-----|
| ①市值 | ≥50亿 | ≥30亿 | ≥2亿AUD |
| ②营收 | ≥10亿 | ≥3亿 | ≥1亿AUD |
| ③毛利率 | 制造业/科技/医药≥15%, 消费≥20%, 金融/矿业跳过 |
| ④ROE | ≥8%, 连续3年<5%剔除, 矿业/成长放宽≥5% |
| ⑤负债率 | 制造业/科技/消费≤65%, 金融/保险跳过, 地产≤75% |
| ⑥估值 | PE历史分位≤70%, PB周期分位≤80%, PS(亏损成长)≤10x |

- **候选池表：** 股票名+代码、环节(带col-tag色标)、市值(核心5只加粗)、毛利率(带色标)、ROE(带色标)、负债率、PE/PB分位、通过数(N/6)
- **观察池表：** 股票名、卡在哪筛(带col-tag)、备注(触发条件)
- **筛除记录表：** 各筛子筛除数+知名示例，末行剩余数加粗绿色
- L7结论：highlight-box写筛选力度、是否需加第七筛、哪些值得关注

### L8 tab

- **产业链（卡片式）：** flex三列布局（上游/中游/下游），上游1-2层(色块+公司+下箭头)，中游含主链横向流程(红底亏损提示)+辅材/分支，下游含应用环节+终端用户。底部利润最厚环节提示
- **环节汇总表：** 环节(色标)、代表公司、毛利率(色标)、集中度、议价能力(色标)
- **市场规模表：** TAM、当前渗透率、近年3年CAGR(色标)
- **生命周期：** summary-box，阶段标签+判断依据
- **增长驱动力表：** 驱动力、类型(结构/周期)、可持续性(长/中/短)、量化指标

### L9 tab

- **市场结构表：** 结构类型(色标)、CR3、格局象限(色标)、新进入者威胁(色标)
- **竞争壁垒表：** 壁垒类型、强度(色标)、可持续性(色标)、具体表现
- **定价权测试：** summary-box，龙头能否涨价、毛利率趋势、ROE趋势
- **驱动力拆解：** highlight-box，收入公式+当前量价组合色标
- **驱动因子表：** 因子名、最新值(色标)、历史区间、当前分位(色标)
- **成本结构：** summary-box，类型(重/轻资产)、敏感性、盈亏平衡点
- **竞争格局总结：** highlight-box

### L10 tab

- **周期位置表：** 库存周期阶段(色标)、产能利用率(色标)、价格分位(色标)、盈利状态(色标)
- **PE悖论检查：** danger-box，提示周期股PE低≠便宜的陷阱
- **估值水平表：** PE/PB当前值、历史中位数、80%分位、20%分位
- **政策环境：** summary-box，方向(色标)+关键政策列表+周期位置(色标)
- **买入清单表：** 5个条件(周期位置/估值/政策/格局/增长)、结果(✅❌⚠️色标)、备注
- **综合判断框：** highlight-box含买入/等待/回避标签、预期回报、仓位建议(色标)、跟踪指标、买入策略(分段标注)
- **卖出信号：** danger-box，4个编号卖出信号(圆角圆形编号)
