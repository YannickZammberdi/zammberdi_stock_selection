# 股票分析报告模板

## 用途

为一只股票生成带tab导航的HTML分析报告，内容包括总结论、L1三张报表、L2关键指标+排雷、L3估值、L4杜邦分析、L5行业分类，各放在独立tab中。覆盖A股/ASX/美股，根据行业类型（周期/成长/防御/价值/保险/勘探）调整框架。

## 参考模板

参考 `雅克科技_002409_分析报告.html` 的HTML结构和CSS样式。模板已包含完整的CSS、tab-bar和JS交互，直接复制后替换数据和内容即可。

## 输出路径

`stock-learning/stocks-analysis/{股票简称}_{股票代码}_分析报告.html`

> 示例：`雅克科技_002409_分析报告.html`

## 报告结构

### 页面布局

```
body (bg:#f5f5f5, padding:0)
  .report-wrap (max-width:800px, margin:0 auto, bg:#fff)
    h1 + .subtitle + .meta
    .tab-bar (sticky top, flex)
    .tab-content#tab-summary > .tab-inner
    .tab-content#tab-l1 > .tab-inner
    .tab-content#tab-l2 > .tab-inner
    .tab-content#tab-l3 > .tab-inner
    .tab-content#tab-l4 > .tab-inner
    .tab-content#tab-l5 > .tab-inner
    .tab-content#tab-l6 > .tab-inner
    .footer
```

### Tab导航

| 按钮 | data-tab | 内容 |
|------|----------|------|
| 总结论 | summary | 公司基本画像 + 综合诊断 + verdict |
| L1 三张报表 | l1 | 利润表 + 资产负债表 + 现金流量表 |
| L2 关键指标+排雷 | l2 | 关键指标表 + 分业务分析 + 排雷清单 |
| L3 估值 | l3 | 估值水平 + PE Band + 总结 |
| L4 杜邦分析 | l4 | 杜邦拆解 + 三年趋势 + 赚钱模式判断 |
| L5 行业分类 | l5 | 行业类型 + 集中度/壁垒 + 生命周期 + 估值匹配 + 成色检验 |
| L6 综合实战 | l6 | 六步法总结 + 好公司三要素 + 风险评估 + 前景展望 + 讨论问题 |

每个tab-content内层套`.tab-inner { padding: 0 12px; }`，使所有内容有统一的水平间距。

### CSS准则

- 所有CSS样式与参考模板完全一致
- h2: font-size 18px, margin 28px 0 12px, border-bottom 2px #1a1a1a
- h3: font-size 15px, margin 18px 0 8px
- table: width 100%, border-collapse, th bg #f5f5f5, 偶数行 bg #fafafa
- 结论框: summary-box(浅黄+橙色左边框), danger-box(浅红+红色左边框), highlight-box(浅蓝+蓝色左边框)
- verdict: 居中圆角框, verdict-ok(绿)/verdict-caution(橙)/verdict-bad(红)
- 评分表加在verdict上方，左列为维度名，右列为得分和说明，总分行加粗
- h1/title无margin水平值, subtitle/meta居中
- meta中日期统一使用 `YYYY-MM-DD` 格式（如 `分析日期：2026-06-29`），精确到日

### JS交互

```javascript
// Tab切换逻辑
var btns = document.querySelectorAll('.tab-btn:not(.disabled)');
var contents = document.querySelectorAll('.tab-content');
btns.forEach(function(btn) {
  btn.addEventListener('click', function() {
    btns.forEach(function(b) { b.classList.remove('active'); });
    contents.forEach(function(c) { c.classList.remove('active'); });
    btn.classList.add('active');
    var tab = btn.getAttribute('data-tab');
    document.getElementById('tab-' + tab).classList.add('active');
  });
});
```

## 内容填充规则

### 总结论tab

- **公司基本画像表：** 名称、代码、行业类型(L5分类)、地位、营收、净利润、CFO、总资产、净资产、ROE
- **综合评分verdict：** 百分制，综合L1-L4所有因素后打分，非仅ROE。含评分表（维度+得分+说明），然后verdict大框显示总分+等级：
  - ≥80分 → verdict-ok（绿色）："品质优秀"
  - ≥60分 → verdict-caution（橙色）："一般关注"
  - <60分 → verdict-bad（红色）："不及格"
- **综合诊断：** 用summary-box，列核心优势+核心风险，各3-4条

#### 评分权重参考

| 维度 | 权重 | 评分依据 |
|------|------|----------|
| ROE | 15分 | ≥15% 15分, ≥10% 10分, ≥5% 5分, <5% 0分 |
| 杜邦驱动质量(ROE来源) | 5分 | 高利润型 5分, 高周转型 4分, 均衡 3分, 高杠杆型 1分, 全靠杠杆且利润差 0分 |
| 利润含金量(CFO/净利) | 15分 | ≥1.0 15分, ≥0.7 10分, ≥0.5 5分, <0.5 0分 |
| 营收增长 | 10分 | ≥10% 10分, ≥5% 7分, ≥0% 4分, <0% 0分 |
| 利润增长 | 10分 | ≥15% 10分, ≥5% 7分, ≥0% 4分, <0% 0分 |
| 负债率 | 10分 | <40% 10分, <55% 7分, <70% 4分, ≥70% 0分 |
| 短期偿债(货币资金-短期有息) | 10分 | 充裕覆盖 10分, 基本覆盖 7分, 缺口<30% 4分, 缺口大 0分 |
| 毛利率水平与趋势 | 10分 | ≥30%且稳定 10分, ≥15%且稳定 7分, ≥10% 4分, <10%或向下 0分 |
| 估值(PE vs 合理区间) | 10分 | PE在合理区间 10分, 偏高<30% 7分, 偏高<50% 4分, 严重偏高0分 |
| FCF | 5分 | 持续为正 5分, 偶尔为负 3分, 持续为负 0分 |

保险等特殊行业可调整部分维度权重和依据，但框架保持一致。评分表放在verdict上方。

### L1 tab

- **利润表：** 营收、净利润、扣非、毛利率、净利率、费用率、Q4营收/Q4净利，含同比和评价列
- **资产负债表：** 货币资金、应收、存货、短借、长借、净资产、负债率，含评价列
- **现金流量表：** CFO、CFI、CFF、CFO/净利润、FCF，含评价列
- 每张表后跟一个结论框（summary-box 或 highlight-box）

### L2 tab

- **关键指标表：** ROE、毛利率、净利率、营收增长、利润增长、CFO/净利润、负债率、分红率，含好公司标准和评价列
- **分业务分析：** 各板块营收、占比、增长、毛利率及评价
- **排雷清单：** 风险项+严重程度等级(红/黄/绿)+依据，正面项单独列出

### L3 tab

- **当前估值表：** 股价、总市值、PE(TTM)、PE(TTM扣非)、PE(2026E)、PB、PS、PCF、股息率
- **PE Band区间：** 悲观/保守/中性/券商目标/当前价/历史高位六种情景
  - 合理PE取值：ROE>20%→20-30x，ROE 15-20%→15-25x，ROE 10-15%→10-18x，ROE<10%→<10x
- **历史PE区间：** 最大值/80%分位/50%分位/20%分位/最小值 + 当前PE对比
- **估值总结：** 用danger-box(估值过高)或highlight-box(估值合理)，给出具体建议

### L4 tab

- **杜邦拆解表：** 净利润率、资产周转率、权益乘数、ROE，含近三年趋势列
- **赚钱模式判断：** 用highlight-box判断公司属于高利润型/高周转型/高杠杆型，并给出评价
- **去杠杆ROA（净利润率×周转率）：** 判断去掉杠杆后的真实基础回报
- **资产回报效率：** 每1元总资产产生多少净利润，与同行对比
- **核心结论：** 用summary-box写出杜邦分析的核心发现——ROE靠什么驱动，趋势是否健康

### L5 tab

- **行业类型判断：** 用summary-box写明该行业属于哪一类（周期性/成长性/防御性/价值型），并给出判断依据
  - 纯周期（钢铁/煤炭/水泥/养殖/航运/证券/基础化工）vs 混合周期（半导体/光伏/锂矿/汽车）：各有自身周期长度和驱动因素
  - 纯周期 = 行业成熟，长期增速≈0或仅随GDP，利润靠供需错配
  - 混合周期 = 有长期成长趋势 + 阶段性供需波动
  - ✅ 参考 `materials/L5-industry.md` 四大分类和N5中的细分

- **行业集中度与壁垒：** 判断市场格局（CR3>70%高度集中/分散/正在集中），列出核心进入壁垒类型（品牌/技术/牌照/规模/网络效应/渠道）

- **生命周期阶段：** 导入期/成长期/成熟期/衰退期 → 决定投资策略重点

- **当前周期位置（周期股适用）：** 判断行业当前处于什么周期位置，用danger-box或highlight-box提示
  - 周期底部（亏损/停产/库存出清）→ PB<1时可能是机会
  - 周期顶部（满产满销/暴利）→ PE极低时可能是风险

- **估值方法匹配：** 检查当前市场使用的估值方法是否正确
  | 行业类型 | 首选估值 | 辅助指标 |
  |----------|---------|---------|
  | 成长型（科技/医药） | PS / PEG | PE（盈利后），看营收增速 |
  | 周期型（钢铁/化工） | PB（底部）| PE（顶部参考），看产能利用率 |
  | 防御型（公用事业） | PE / 股息率 | EV/EBITDA，看现金流稳定性 |
  | 金融（银行/保险） | PB | PE，看不良率(银行)/UITR(保险) |
  | 消费（白酒/家电） | PE | 毛利率/ROE，看品牌力 |

- **成长股成色检验（可选）：** 对高估值成长股（PS>10或PE>50），使用四步验证法
  1. 营收—有没有实在的生意（营收>20亿且增速>20% vs <10亿但市值>100亿）
  2. 毛利率—产品竞争力（>30%且稳定 vs 极低或营收太少时无意义）
  3. 经营现金流—能不能自己造血（CFO为正且在改善 vs 持续为负靠融资续命）
  4. PS—梦想溢价（<10合理, 10-20需高增长支撑, >20预警, >50纯投机）

### 特殊行业框架覆盖

| 公司类型 | 适用框架 | 不适用项 |
|----------|---------|---------|
| 保险 | L1-L4调整版（GWP/UITR/CET1替代标准指标） | 标准L1部分指标 |
| 银行 | 标准L1-L4有效，关注净息差/不良率/拨备覆盖率 | — |
| 地产 | L1-L4有效，关注NAV/销售回款/负债率 | — |
| 纯周期股（钢铁/养殖） | L1-L4有效，但估值看PB非PE | L3 PE Band需备注周期失真 |
| 成长初期/情绪股 | L1-L4有效，但需配合L5四步验证法 | L3 PE无意义时用PS |
| 勘探期矿业 | 非L1-L4框架：项目管线/现金跑道/矿权基础/股权稀释 | 标准L1-L4基本不适用 |

### L6 tab

#### 1. 六步法总结

用summary-box呈现六步法核心结论，每条1行：

```
Step 1 - 行业定位：{行业类型、集中度、壁垒类型}
Step 2 - 财务体检：{ROE水平、杜邦类型、CFO质量}
Step 3 - 估值判断：{PE/PB、历史分位、与同行对比}
Step 4 - 风险排查：{核心风险点}
Step 5 - 结论：{好公司？价格合理？}
Step 6 - 跟踪：{关键指标、卖出信号}
```

#### 2. 好公司三要素检查

用table呈现：

| 要素 | 判断 | 依据 |
|------|------|------|
| ① 好行业 | ✓/✗ | 需求、壁垒、格局评估 |
| ② 好公司 | ✓/✗ | ROE>15%，毛利率稳定或提升，CFO>净利润 |
| ③ 好价格 | ✓/✗ | PE/PB在历史偏低分位，PEG<1.5 |

#### 3. 风险评估

##### a. 三个陷阱排查

用danger-box标出命中的陷阱，都不命中也说明：

```
☐ 低PE + 低增长 + 高负债 = 价值陷阱 → 命中/未命中
☐ 低PE + 周期顶点 = 周期陷阱 → 命中/未命中
☐ 高PE + 高增速但现金流为负 = 烧钱陷阱 → 命中/未命中
```

##### b. 管理层与治理风险

用table评估：

| 事项 | 判断 | 依据 |
|------|------|------|
| 管理层能力与履历 | 好/一般/差 | 过往业绩、行业经验 |
| 股权结构与利益一致性 | 强/一般/弱 | 大股东持股、管理层持股 |
| 关联交易/暴雷历史 | 无/有但可控/严重 | 历史违规、监管处罚、诉讼 |
| 资本配置记录 | 合理/激进/混乱 | 分红率、并购质量、回购节奏 |
| CEO交接/关键人员变动 | 无风险/需关注/高风险 | 交接期、新CEO背景 |

##### c. 盈利可持续性

用table列出利润质量拆解：

| 维度 | 判断 | 依据 |
|------|------|------|
| 增长靠量还是靠价？ | 量驱动/价驱动/混合 | 量价拆解 |
| 涨价能否持续？ | 能/需观察/不能 | 定价权来源、竞争反应 |
| 非经常性利润占比 | 低/中/高 | 占净利润比例 |
| 大客户/大项目集中度 | 分散/集中/极度集中 | 前5大客户占比 |
| 政策/监管依赖度 | 低/中/高 | 补贴、牌照、准入管制 |

##### d. 关键变量敏感性

识别2-3个最可能改变公司基本面的变量，每个变量写三情景：

```
变量1: {描述}
  乐观: {条件} → {对EPS影响}
  中性: {条件} → {对EPS影响}
  悲观: {条件} → {对EPS影响}
```

#### 4. 前景展望

##### a. 未来1-3年核心驱动

用summary-box列出正向驱动和负向拖累，各2-3条：

```
正向驱动：
1. {因素} — {逻辑}
2. {因素} — {逻辑}

负向拖累：
1. {因素} — {逻辑}
2. {因素} — {逻辑}
```

##### b. 三情景简表

| 情景 | 假设条件 | EPS预估 | PE区间 | 预期回报 |
|------|---------|---------|--------|---------|
| 乐观 | {触发条件} | {预估EPS} | {预估PE} | {预期回报} |
| 中性 | {触发条件} | {预估EPS} | {预估PE} | {预期回报} |
| 悲观 | {触发条件} | {预估EPS} | {预估PE} | {预期回报} |

附概率判断（如"中性概率50%，乐观30%，悲观20%"）。

#### 5. 讨论问题（阅读引导）

写在L6 tab底部，不作强制回答：

```
讨论问题（阅读后可以聊这些）：
1. 这笔生意你看懂了吗？一句话说清这家公司怎么赚钱？
2. 你最大的担忧是什么？哪个数据让你睡不着？
3. 如果现在买入，期待的回报来源是什么？
4. 什么情况下你会坚决卖出？
```
