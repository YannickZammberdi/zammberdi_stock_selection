# 个股交易计划模板

## 用途

为一只**基本面已过关**的个股（建议只对持仓中前20%的精选标的使用）生成L11-L13交易计划。输入股票代码，输出一张A4纸篇幅的HTML交易计划书。

## 参考模板

参考 `新和成_002001_交易计划.html` 的HTML结构和内容布局。

## 输出路径

```
stock-learning/stocks-analysis/{股票简称}_{股票代码}_交易计划.html
```

> 示例：`新和成_002001_交易计划.html`

## 报告结构

单页，无tab，全部展开显示。包含以下模块：

### 1. 头部信息
- 股票名称代码、日期（YYYY-MM-DD格式，精确到日）、L5分类
- 当前状态标签（计划中/持有中/已结束）

### 2. 估值概览（L11）
- 赚什么钱（估值修复/业绩增长/股息）
- 当前PE分位、PB分位、ROE
- 合理PE目标
- PE分位进度条

### 3. 估值目标
- 现价、EPS(TTM)
- PE修复到不同目标的空间（%）
- 时间预期

### 4. 买入检查清单（正反两面）
- 左列：利好因素（绿色，打勾）
- 右列：风险因素（红色，感叹号）
- 底部：综合判断 + 建议

### 5. 仓位计划（L12）
- 总计划仓位 / 已买 / 剩余
- 分批建仓明细（第一批/第二批/第三批的价格和条件）

### 6. 止损规则（L13）
- 硬止损价和幅度（%）
- 基本面止损条件列表

### 7. 止盈规则（L13）
- 各批次止盈触发条件（PE目标价或股价）
- 移动止盈规则

### 8. 跟踪指标
- 每行一个指标：名称 + 用途说明 + 当前值 + 信号（红/绿/黄）

### 9. 持有期限
- 最短持有时间 + 最长持有时间
- 三种情景：估值先到 / 业绩兑现 / 逻辑证伪

### 10. 催化剂
- 左列：利好催化剂
- 右列：利空风险

## 数据来源

- 股价/PE/PB：东方财富、亿牛网等实时行情
- PE/PB历史分位：亿牛网（分位数据）
- VA/VE/蛋氨酸价格：券商研报、百川盈孚
- 财报数据：已生成的L1-L5分析报告

## 使用条件

该技能**仅用于已经通过L1-L5基本面分析、值得进一步考虑的标的**。不适用于：
- 纯概念炒作股
- 尚未盈利的成长初期公司
- 已触发卖出信号的持仓

## 用户角色

- 用户负责提供：股票代码 + 当前估值数据 + 个人仓位上限偏好
- 模型负责：生成完整HTML交易计划书

## 样式规范

CSS全部内联在HTML中，无需外部依赖。配色沿用已有的`col-tag`体系（绿/红/黄/蓝），新增：
- `bar` 进度条（PE分位可视化）
- `check` 检查清单（圆点+打勾/感叹号）
- `grid-3` 三列仓位/止损/止盈卡片

参考现有 `新和成_002001_交易计划.html` 的全部样式和结构，生成时替换数据和内容即可。
