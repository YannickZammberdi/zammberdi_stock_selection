# L3 - 估值指标

这一章讲如何给股票"定价"——你现在买进，是贵了还是便宜了？

---

## 一、PE（市盈率）

> 公式：股价 ÷ 每股收益（EPS），或总市值 ÷ 净利润

**最常用的估值指标。**

### 三种PE

| 类型 | 含义 | 使用场景 |
|------|------|----------|
| 静态PE | 用上一年度净利润 | 滞后，但确定性强 |
| 滚动PE（TTM） | 用过去四个季度的净利润 | 最常用，反映最新状况 |
| 动态PE | 用预测的未来净利润 | 前瞻性，但靠猜 |

### PE怎么看

| PE范围 | 含义 | 常见情形 |
|--------|------|----------|
| 0-10 | 极低 | 周期股低谷期，或被市场抛弃 |
| 10-20 | 中等偏低 | 成熟行业、价值股 |
| 20-30 | 中等偏高 | 稳定增长型公司 |
| 30-50 | 高 | 高成长公司，市场愿意给溢价 |
| 50+ | 极高 | 热门成长股或泡沫 |

**PE是双向的：**

> PE高 → 可能被高估，也可能是因为未来高增长预期
> PE低 → 可能是便宜，也可能是公司前景有问题（"价值陷阱"）

### PE的局限

- 净利润为负时，PE无意义（亏损公司看PS）
- 周期性行业净利润波动大，PE会剧烈失真（钢铁股利润高点PE=3，低点PE=50+）

---

## 二、PB（市净率）

> 公式：股价 ÷ 每股净资产，或总市值 ÷ 净资产

**适用于：** 银行、保险、房地产等重资产行业。

### PB怎么看

| PB | 含义 |
|----|------|
| <1（破净） | 市值低于公司清算价值，理论上"便宜" |
| 1-3 | 正常范围 |
| 3-5 | 溢价，说明市场认可其盈利能力超出净资产 |
| >5 | 高溢价，通常出现在轻资产高ROE公司 |

**实战案例：**

```
贵州茅台：PB ≈ 9（市场认可其品牌价值远超账面净资产）
工商银行：PB ≈ 0.5-0.7（长期破净，市场对不良贷款有担忧）
```

---

## 三、PS（市销率）

> 公式：总市值 ÷ 营收

**适用于：** 净利润为负的公司（如早期互联网企业、创新药企）。

- PS低说明市场给每单位营收的定价便宜
- 但PS没有考虑成本和利润，不能只看PS

---

## 四、PCF（市现率）

> 公式：总市值 ÷ 经营现金流

**为什么重要：** 经营现金流比净利润更难造假，PCF比PE更能反映真实估值。

---

## 五、PEG（市盈率相对盈利增长比率）

> 公式：PE ÷ 净利润增长率（%）

**目的是把估值和成长性结合起来，适用于成长股。**

| PEG | 含义 |
|-----|------|
| <1 | 可能被低估（增长高，PE低） |
| =1 | 估值合理 |
| >1 | 可能被高估（增长跟不上PE） |

**举例：**

```
公司A：PE=30，增速=30% → PEG=1（合理）
公司B：PE=50，增速=20% → PEG=2.5（偏贵）
公司C：PE=10，增速=5%  → PEG=2（平庸，看起来便宜其实不便宜）
```

---

## 六、股息率

> 公式：每股分红 ÷ 股价

**适用于：** 防御性投资、收息型公司。

| 股息率 | 含义 |
|--------|------|
| >5% | 高股息，但需警惕是否可持续 |
| 3-5% | 良好，常见于公用事业、银行 |
| 1-3% | 一般，成长股通常不分红或低分红 |
| 0% | 不分红（利润留作再投资） |

> 高股息率 ≠ 好投资。如果公司股价腰斩，股息率会被动升高——可能是陷阱。

---

## 七、EV/EBITDA（企业价值倍数）

> 公式：企业价值（EV）÷ 息税折旧摊销前利润（EBITDA）

**为什么用EV而不是市值：** EV包含了负债，反映了收购这家公司要花多少钱。

**适用于：** 高速公路、电力、电信等重资产高负债行业。

| 范围 | 含义 |
|------|------|
| <8 | 偏低 |
| 8-15 | 正常 |
| >15 | 偏高 |

---

## 八、如何综合运用

**不同行业关注不同的核心估值指标：**

| 行业 | 首选估值 | 其次 |
|------|---------|------|
| 消费/医药 | PE | PS（亏损时） |
| 银行/保险 | PB | PE |
| 周期股（钢铁/化工） | PB（低点） | PE（高点参考） |
| 科技/互联网 | PS / PE | PEG |
| 公用事业 | 股息率 | EV/EBITDA |

### 三个核心判断维度

**1. 跟历史比——当前估值处于什么分位？**

```
茅台PE历史区间：25-70
当前PE=30 → 处于历史低位区间
→ 如果基本面没问题，可能是买入时机
```

**2. 跟同行比——比竞争对手贵还是便宜？**

```
白酒行业：
  茅台PE=30，五粮液PE=20，泸州老窖PE=22
  茅台更贵，但毛利率/ROE也更高
→ 贵有贵的道理
```

**3. 跟自身增速比——PEG是否合理？**

```
公司PE=40，增速只有10% → PEG=4（太贵）
公司PE=20，增速15% → PEG≈1.3（合理偏贵一点）
```

---

## 估值陷阱速查

| 陷阱 | 说明 |
|------|------|
| 低PE陷阱（价值陷阱） | PE低可能是因为公司前景很差，不是真便宜 |
| 高PE≠泡沫 | 高成长公司高PE是正常的，关键是增速能否持续 |
| 周期股PE失真 | 利润高点PE最低（恰恰是最贵的时候），利润低点PE最高 |
| 只看绝对值 | 必须结合历史分位+同行对比 |
