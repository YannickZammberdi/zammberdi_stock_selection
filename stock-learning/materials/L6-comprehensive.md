## 一、完整的分析框架

### 六步法

```
Step 1: 行业定位
  → 属于什么行业？周期/成长/防御/价值？
  → 行业集中度？进入壁垒？
  （回忆L5）

Step 2: 财务体检
  → 看ROE：高不高？靠什么驱动？（L4杜邦分析）
  → 看毛利率+净利率：趋势向上还是向下？（L2）
  → 看经营现金流：和净利润匹配吗？（L1）

Step 3: 估值判断
  → 用哪个指标？（PE/PB/PS/PEG）
  → 在历史什么分位？
  → 和同行比贵不贵？（L3）

Step 4: 风险排查
  → 资产负债表有没有雷？（应收/商誉/高负债）
  → 现金流有没有问题？
  → 行业有没有结构性风险？

Step 5: 给出结论
  → 好公司还是一般公司？
  → 当前价格合理吗？
  → 如果买，你的买入逻辑是什么？

Step 6: 持续跟踪
  → 关键指标是什么？什么信号会改变你的判断？
```

## 二、实战案例演示

### 案例：分析某白酒公司

**Step 1 - 行业定位**

```
行业：白酒（消费品）
类型：偏防御性+消费属性
集中度：高度集中，CR3（茅五洋）占高端大部分
壁垒：品牌壁垒极强，新品牌几乎不可能挑战龙头企业
```

**Step 2 - 财务体检**

```
ROE：33%（近5年均>25%），优秀
杜邦拆解：
  净利润率 33% ← 核心驱动力（高利润型）
  资产周转率 0.6次
  权益乘数 1.5

毛利率：76%（连续5年稳定在75-78%）
经营现金流/净利润：1.2（现金质量好）
营收增速：15%（稳定增长）
```

**Step 3 - 估值判断**

```
当前PE：22
历史PE区间：20-60
当前处于历史30%分位（偏低）
同行对比：茅台PE=30，五粮液PE=22，泸州老窖PE=22
→ 处于行业中低水平
```

**Step 4 - 风险排查**

```
资产负债率：25%（很低，安全）
应收账款：几乎为0（先款后货）
商誉：极低
存货：基酒库存，不贬值反而升值
→ 资产负债表干净，无明显风险
```

**Step 5 - 结论**

```
好公司 ✓
当前PE=22处于历史偏低位置 ✓
财务质量极好 ✓
→ 如果对白酒行业前景有信心，当前估值有安全边际
```

**Step 6 - 跟踪**

```
关键跟踪指标：毛利率是否稳定、合同负债（预收款）是否下降
卖出信号：营收连续2个季度下滑、毛利率跌破70%
```

## 四、实战模板要点速记

**判断好公司的三要素：**

```
① 好行业 → 需求稳定or增长，壁垒高，格局好
② 好公司 → ROE>15%，毛利率稳定或提升，经营现金流>净利润
③ 好价格 → PE/PB在历史偏低分位，PEG<1.5
```

**警惕"看起来便宜"的陷阱：**

```
低PE + 低增长 + 高负债 = 价值陷阱
低PE + 周期顶点 = 周期陷阱
高PE + 高增速但现金流为负 = 烧钱陷阱
```

**卖出信号：**

```
- 基本面恶化（毛利率下降、现金流变差）
- 估值极度高估（PE处于历史90%分位以上）
- 找到更好的标的
```
