## 二、六步法参考（报告L6 Tab内容）

每次报告的L6 Tab会包含以下四个板块：

### 六步法总结

| 步骤 | 核心问题 | 输出 |
|------|---------|------|
| Step 1 行业定位 | 类型？集中度？壁垒？ | L5 Tab汇总 |
| Step 2 财务体检 | ROE多少？靠什么驱动？现金流健康？ | L1+L2+L4 Tab汇总 |
| Step 3 估值判断 | PE/PB在历史什么位置？ | L3 Tab汇总 |
| Step 4 风险排查 | 资产负债有没有雷？ | 各Tab反推 |
| Step 5 结论 | 好公司？价格合理？ | 总结论Tab |
| Step 6 跟踪 | 看什么指标？什么信号卖出？ | 总结论Tab |

### 好公司三要素检查

```
① 好行业 → 需求稳定or增长，壁垒高，格局好
② 好公司 → ROE>15%，毛利率稳定或提升，CFO>净利润
③ 好价格 → PE/PB在历史偏低分位，PEG<1.5
```

### 三个陷阱排查

```
低PE + 低增长 + 高负债 = 价值陷阱
低PE + 周期顶点 = 周期陷阱
高PE + 高增速但现金流为负 = 烧钱陷阱
```

### 讨论问题（阅读后引导）

写在报告L6 Tab底部。不是要求回答，而是引导阅读后思考：

```
讨论问题（阅读报告后，我们可以聊这些）：
1. 这笔生意你看懂了吗？一句话说清这个公司怎么赚钱？
2. 你最大的担忧是什么？哪个数据让你睡不着？
3. 如果现在买入，期待的回报来源是什么？（估值回归/业绩增长/两者兼有？）
4. 什么情况下你会坚决卖出？
```
