# L12 - 仓位管理与分批建仓

> L11告诉你**现在贵不贵**，L12告诉你**买多少、怎么买**。

---

## 一、核心原则

```
单只股票的仓位 = 你的信心程度 + 当前估值位置
```

- **永远不要全仓一只票**（哪怕你100%确定）
- **永远不要"一把梭"**（分批买是为了应对"你以为的低点不是最低点"）
- **仓位大小决定你的心态**——买多了拿不住，买少了赚了也没意义

---

## 二、仓位上限：先定死规则

### 按资金量分

| 总资金 | 单只上限 | 同时持有上限 | 理由 |
|--------|---------|-------------|------|
| < 10万 | 30% | 3-5只 | 资金少，需要集中才能有收益 |
| 10-50万 | 20% | 5-8只 | 适度分散，控制单票风险 |
| 50万+ | 15% | 8-12只 | 充分分散，机构级风控 |

### 按信心度分（在总资金上限内）

| 信心等级 | 条件 | 仓位 |
|---------|------|------|
| ⭐⭐⭐ | L1-L5基本面优秀 + L11估值低分位 + 你自己能说清赚什么钱 | 上限的100%（如20%） |
| ⭐⭐ | 基本面好但估值中位，或有不确定性 | 上限的50%（如10%） |
| ⭐ | 还在研究/观察阶段 | 不超过5%，或放入观察池不买 |

---

## 三、分批建仓：三批法

不要一次买完。分三批，每批间隔5-10%的跌幅（或关键事件触发）。

### 标准三批

```
第一批 40%仓位（发现机会时）
   ↓ 等股价跌5-10%，或等季报确认
第二批 35%仓位（验证逻辑后）
   ↓ 等股价再跌5-10%，或等新催化剂
第三批 25%仓位（最后确认后）
```

### 变体：按估值分位

| 当前PE分位 | 建议动作 | 买多少 |
|-----------|---------|--------|
| < 20% | 可以一次性买入上限的80% | 估值极端低位，不需要太分散 |
| 20-40% | 分2-3批买入 | 标准三批 |
| 40-60% | 只买第一批观察仓（上限的30%） | 不着急 |
| > 60% | 不买 | 等回调 |

### 极端情况：如果买了就涨

```
第一批买了 → 股价涨20%以上
→ 停止加仓（不要追高）
→ 如果涨到估值>60%分位，连第一批也卖掉
```

---

## 四、加仓规则（L12最易犯的错）

### 正确的加仓逻辑

```
加仓条件（需同时满足）：
1. 股价比上次买入跌了10%以上
2. 买入逻辑没有变（基本面没恶化）
3. 估值分位更低了（PE/PB比上次更低）

只有三个条件都满足才加
```

### 错误的加仓逻辑

```
❌ "已经涨了10%，追进去怕踏空"
❌ "跌了20%了，补仓拉低成本"（这叫摊平，不叫加仓）
❌ "出了个利好新闻，再加点"（先看利好是否改变基本面）
```
