# L0-B1 - 澳洲宏观与大宗商品

## ASX底色与A股完全不同

A股的驱动力是 **国内政策**，ASX的驱动力是 **全球利率 + 大宗商品**。

ASX200 的成分构成决定了这件事：

| 板块 | 权重 | 特征 |
|------|------|------|
| 金融（四大行+保险） | ~30% | 受RBA利率+房贷周期驱动 |
| 矿业（BHP/RIO/FMG等） | ~20% | 受铁矿石/金价驱动 |
| 医疗（CSL/COH/RMD等） | ~10% | 受全球需求和汇率影响 |
| 其他 | ~40% | 分散 |

**所以ASX的宏观判断要看三个东西：**
1. RBA利率 → 金融板块+整体市场水位
2. 大宗商品价格 → 矿业板块
3. 中国经济 → ASX最大的外部变量

---

## 一、RBA（澳洲联储）利率

### 影响机制

| RBA利率 | 对ASX的影响 |
|---------|------------|
 | 降息周期 | 利好银行（贷款需求增加）+ 利好整体估值 |
 | 加息周期 | 利空银行（贷款减少）+ 压低整体估值 |
 | 维持不变 | 中性，取决于市场预期 |

### 如何跟踪

| 时间 | 事件 |
|------|------|
| 每月第一个周二 | RBA利率决议 |
| 每季度 | RBA货币政策声明 + 经济预测 |
| 不定期 | RBA行长讲话 |

**利率方向判断：**

```
RBA在降息 → ASX整体偏多（尤其是金融股 + 高估值成长股）
RBA在加息 → ASX整体承压（防御性 > 成长性）
RBA停止加息但还没降 → 观望期，市场波动
```

### 关注点不只是利率本身

**RBA对通胀和就业的描述比利率本身更重要。** 例如：

- "通胀仍然过高，不排除进一步加息" → 鹰派，利空
- "通胀在向目标回归" → 鸽派，利好
- "劳动力市场仍然紧张" → 可能维持高利率更久

> 月频做法：每月第一个周二关注RBA利率决议结果 + 声明摘要。

## 二、大宗商品：ASX矿业的风向标

ASX矿业公司的盈利直接绑定大宗商品价格。

### 对ASX最关键的商品

| 商品 | 驱动因素 | 影响的公司 |
|------|---------|-----------|
| 铁矿石 | 中国房地产+基建+钢铁产量 | BHP, RIO, FMG |
| 金价 | 全球避险情绪+实际利率+美元 | NEM, EVN, NST |
| 锂 | 全球电动车渗透率+供给 | PLS, IGO, LTR |
| 煤炭 | 全球能源价格 | WHC, YAL, NHC |
| 铜 | 全球电气化+基建 | BHP, RIO |

### 怎么看

| 商品价格趋势 | 对ASX的含义 |
|-------------|------------|
| 铁矿石上涨 | ASX200最强（矿业权重股大涨） |
| 金价上涨 | 防御性机会，市场避险情绪重 |
| 铁矿石+金价都跌 | ASX大概率走弱 |

> **月频做法：** 每月看一眼铁矿石+金价的月线趋势，记录方向。

## 三、中国经济：ASX最大的外部变量

ASX不是"澳洲经济的股票"，而是 **"卖资源给中国的公司的股票"**。

### 中国指标与ASX的对应关系

| 中国指标 | 对ASX的影响 |
|---------|------------|
| 中国PMI > 50 | 铁矿石需求强劲 → 矿业股涨 → ASX涨 |
| 中国房地产投资回升 | 铁矿石+煤炭需求 → 矿业股涨 |
| 中国社融大超预期 | 经济刺激预期 → ASX资源股大涨 |
| 中国经济低于预期 | ASX资源股承压 |

> ASX投资者每天早上都在看中国的PMI和房地产数据。**这比RBA利率更频繁地影响ASX。** 所以你的L0-A1关于中国PMI/社融的判断，对ASX也同样重要。

## 四、AUD/USD 汇率

| AUD/USD 走势 | 含义 |
|-------------|------|
| 上涨 | 澳元走强 → 利好进口但利空出口（矿业公司收入换成澳币后缩水） |
| 下跌 | 澳元走弱 → 利好矿业出口收入 → 但增加输入性通胀 |
| 正常范围 | 0.63~0.70（近年常态） |
| >0.70 | 澳元偏强 |
| <0.63 | 澳元偏弱 |

> 汇率变动对ASX的矿业股有直接的盈利影响。**澳元弱时，BHP/RIO等以美元计价的收入换成澳币后更多。**

## 五、综合判断

| ASX环境 | 对应条件 | 怎么做 |
|---------|---------|-------|
| 强势 | RBA降息 + 大宗商品涨 + 中国经济好 | 重仓，攻守兼备 |
| 温和 | RBA稳定 + 商品横盘 | 正常操作，精选个股 |
| 弱势 | RBA加息 + 商品跌 + 中国经济弱 | 轻仓或防御（医疗+现金） |

**更新节奏：**
- RBA利率决议日（每月第一周二）：跟踪
- 大宗商品月线：月底看一眼
- 中国PMI（每月1号）：自动关联L0-A1
