# L7 - 量化初筛框架

## 用途

在跑完整的L1-L6之前，先用量化指标快速筛选。输入一个**市场 + 一个行业**，输出**5-10只候选股**（行业小的3只也行），让L1-L6只用在值得深度分析的标的上。

## 输入 → 漏斗 → 输出

```
市场 + 行业 → ①市值 → ②营收 → ③毛利率 → ④ROE → ⑤负债率 → ⑥估值 → 5-10只
```

每轮保留约50-70%的标的。行业大（200+只）就收紧阈值，行业小（30只以下）就放宽或跳过部分筛子，最终留5-10只跑L1-L6。

## 六道筛子

### 第一筛：市值（剔除小票）

| 市场 | 标准 | 理由 |
|------|------|------|
| A股主板 | <50亿剔除 | 小票流动性差，操纵风险高，机构不覆盖 |
| A股创业板/科创板 | <30亿剔除 | 适当放宽，但仍有退市风险 |
| ASX | <2亿澳元剔除 | 大量小矿企和空壳公司，暴雷率极高 |

> 这步能筛掉约30-40%。市值是公司质量最简单的代理变量。A股历史上退市的公司，退市前市值低于30亿的比例极高。

### 第二筛：营收（剔除太小的生意）

| 市场 | 标准 | 理由 |
|------|------|------|
| A股主板 | <10亿剔除 | 连10亿生意都没有，很难撑起50亿+市值 |
| A股创业板/科创板 | <3亿剔除 | 允许成长初期公司，但3亿是经营存续底线 |
| ASX | <1亿AUD剔除 | 同上 |

> 注意：毛利极低（<10%）的公司，即使营收大也可能是虚胖（如贸易商），需要后续毛利率筛子来排除。

### 第三筛：毛利率（剔除无定价权的公司）

| 行业类型 | 标准 | 理由 |
|---------|------|------|
| 制造业/科技/医药 | <15%剔除 | 没有定价权的制造业很难赚钱 |
| 消费品牌 | <20%剔除 | 品牌消费品不到20%毛利率说明品牌力弱 |
| 金融/银行 | 跳过此筛 | 用净息差/NIM替代 |
| 贸易/零售 | <10%剔除 | 纯贸易毛利率天然低，但<10%就是白忙活 |
| ASX矿业 | 跳过此筛 | 矿业毛利率受大宗价格影响大，不适合固定阈值 |

> ⚠ 这是最容易被"特例"打破的筛子。例如 Amazon 早期毛利率极低，但这需要你用 L5 行业分类来判断。不确定时，宁可留下来用 L1-L6 专门分析。

### 第四筛：ROE（剔除资本回报率低的公司）

| 标准 | 说明 |
|------|------|
| <8%剔除 | 低于沪深300长期回报率（约8-10%），说明资本回报效率不够 |
| 连续3年ROE<5%剔除 | 长期低回报的公司，大概率是资本堆积型生意 |
| ASX矿业/成长初期 | 放宽到>5%或跳过，需要用L5特殊行业框架判断 |

> ROE是巴菲特最看重的指标。长期ROE>15%的公司，股价长期表现明显跑赢ROE<8%的公司。

### 第五筛：负债率（剔除高杠杆风险）

| 行业类型 | 标准 | 理由 |
|---------|------|------|
| 制造业/科技/消费 | >65%剔除 | 高负债+低增长=价值陷阱 |
| 金融/银行/保险 | 跳过此筛 | 金融业本身高杠杆经营，用不良率/偿付能力替代 |
| 地产 | >75%剔除 | 地产天然高杠杆，但>75%在行业下行期危险 |
| ASX公用事业 | >70%剔除 | 基础设施可以有较高负债，但需稳定现金流覆盖 |

> 很多优秀公司负债率很低，所以这个筛子比较安全——误杀好公司的概率低。

### 第六筛：估值（剔除明显贵的）

| 指标 | 标准 | 理由 |
|------|------|------|
| PE（盈利企业） | 历史分位>70%剔除 | 不追高，等回落到合理区间再说 |
| PB（周期/金融） | 历史分位>80%剔除 | 周期股用PB，高位不买 |
| PS（亏损成长股） | >10x剔除 | 营收撑不起市值，梦想溢价太高 |
| PE为负（亏损） | 加入观察但不买入 | 亏损股需要L5-L6判断是否为困境反转 |

> ⚠ 这步最容易误杀。高PE不一定贵（可能是高成长）。这时把"不确定"的保留下来，用L1-L6判断。

## 筛子使用流程

```
打开证券软件选股器（或Excel导入数据）：

ROUND 1: 市值过滤
  A股: 市值 > 50亿  ✅ 保留
  ASX: 市值 > 2亿AUD ✅ 保留

ROUND 2: 营收过滤
  营收 > 10亿（A股主板）/ 3亿（创业板）/ 1亿AUD（ASX）✅ 保留

ROUND 3: 毛利率过滤
  毛利率 > 15%（非金融/非矿业）✅ 保留

ROUND 4: ROE过滤
  ROE > 8%（非矿业/非成长初期）✅ 保留

ROUND 5: 负债率过滤
  负债率 < 65%（非金融/非地产）✅ 保留

ROUND 6: 估值过滤
  PE历史分位 < 70% ✅ 保留 → 进入L1-L6分析池
  PE历史分位 > 70% ⏸ 加入等待池（等回调）
  亏损股 → 单独放观察池
```

## 特殊情况偏差（何时跳过或调整）

| 情况 | 调整 |
|------|------|
| 银行/保险 | 跳过负债率筛，跳过毛利率筛 |
| ASX矿业（BHP/RIO/FMG等） | 跳过毛利率筛，ROE放宽到>5% |
| 亏损但高营收成长（PS<3） | 跳过ROE筛和PE筛，保留观察 |
| 上市不满3年 | ROE筛用可用数据替代，其余照常 |
| 周期股当前处于底部（亏损） | 用PB替代PE估值筛 |

## 工具的实操

**A股：**
- 同花顺 → 选股器 → 条件选股 → 输入上述条件
- 东方财富 → 条件选股 → 输入条件

**ASX：**
- CommSec → 筛选器 → 输入条件
- Morningstar Australia → 股票筛选 → 输入条件
- 或者直接下载ASX 300成分股数据到Excel，用公式排序筛选

## L7输出格式

筛选完成后，对保留下来的3-5只股票，输出一个**候选池表**：

```
候选池（2026年Q3初筛结果）：

| 股票 | 市值 | 营收 | 毛利率 | ROE | 负债率 | PE分位 | 行业 |
|------|------|------|--------|-----|--------|--------|------|
| A    | 800亿| 120亿| 28%    | 15% | 45%    | 42%    | 消费 |
| B    | 300亿| 35亿 | 45%    | 12% | 20%    | 35%    | 医药 |
| C    | 60亿 | 12亿 | 22%    | 9%  | 55%    | 28%    | 制造 |

→ 下一轮：对候选池逐一跑L1-L6
```

完成初筛后，就可以进入你熟悉的L1-L6框架了。
