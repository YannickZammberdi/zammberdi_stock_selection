# L8 - 行业画像与产业链

## 用途

在做任何投资决策之前，先搞清楚"这到底是个什么行业"。L8解决三件事：**这个行业做什么的、有多大、靠什么增长**。

---

## 一、行业定义：先画边界

### 1. 这个行业在解决什么问题？

每个行业存在的理由就是满足某种需求。用一句话说清楚。

| 行业 | 解决什么问题 |
|------|------------|
| 证券 | 连接资金端（投资者）和资产端（上市公司），撮合交易 |
| 光伏 | 把阳光转化为电能 |
| 超市 | 把分散的商品集中到一处，节省消费者挑选时间 |
| 金矿 | 从地下挖出黄金 |

> 定义不清，后面所有的分析都会歪。比如"东方财富"应该按证券分析还是按互联网平台分析？答案是先定义清楚它的业务边界（78%证券+20%基金代销），再分别分析每个板块。

### 2. 行业边界在哪？

| 维度 | 问题 |
|------|------|
| 产品/服务范围 | 包括什么、不包括什么 |
| 地理范围 | 全球/全国/区域 |
| 客户群体 | B端/C端、高端/大众 |

**案例：光伏行业边界**
- 包括：硅料、硅片、电池片、组件、逆变器、电站运营
- 不包括：风电、储能（虽然是相关行业）
- 地理：中国是全球最大市场+最大产能国
- 客户：电站投资商（B端）为主

### 3. 行业代码（方便查数据）

| 市场 | 分类体系 | 示例 |
|------|---------|------|
| A股 | 申万行业分类 | 证券(三级行业) |
| A股 | 证监会行业分类 | 资本市场服务(J67) |
| A股 | 中证行业分类 | 证券 |
| ASX | GICS | Diversified Financial Services |
| 全球 | ICB | 证券经纪 |

> 不同分类体系下同一家公司可能被分到不同行业。写报告时注明用的是哪种分类。

## 二、产业链分析：谁在上下游

### 1. 产业链三步法

**第一步：画出链条**

```
上游 → 中游 → 下游 → 终端用户
```

**第二步：标注每个环节**
- 代表公司（龙头是谁）
- 集中度（CR3高不高）
- 利润水平（谁在吃肉、谁在喝汤）

**第三步：找"利润池"**
- 产业链上哪个环节利润最厚？
- 利润是否在转移？（过去三年谁变好了谁变差了）

### 2. 产业链模板

```
上游：{原材料/设备供应商}
  ├─ 代表性公司
  ├─ 毛利率水平
  └─ 集中度/定价权

中游：{制造商/服务商} ← 我们通常分析的环节
  ├─ 代表性公司
  ├─ 毛利率水平
  └─ 集中度/定价权

下游：{渠道/客户}
  ├─ 代表性公司
  └─ 议价能力

终端用户：{谁最终买单}
```

### 3. 案例：证券行业产业链

```
上游                             中游                             下游
交易所(上证/深证) ──提供交易平台──→ 证券公司 ──提供经纪/投行/资管服务──→ 投资者
监管(证监会)  ────制定规则────→    ├─ 东方财富(互联网)    ├─ 散户（主要）
                                    ├─ 中信证券(综合)       ├─ 机构（公募/私募）
                                    └─ 华泰证券(财富管理)    └─ 企业（融资方）
                                    
利润分配：
- 交易所：稳赚（交易规费），议价能力最强
- 券商：靠成交量吃饭，行业集中度低（CR5<30%）
- 散户：整体亏损（零和博弈）
```

### 4. 产业链分析的要点

| 要点 | 为什么重要 |
|------|-----------|
| 利润集中在哪个环节 | 选对环节比选对公司更重要（比如光伏产业链硅料赚最多时，组件商可能不赚钱） |
| 集中度差异 | 上游CR3>70%但下游极其分散 → 上游有强定价权 |
| 环节间的议价能力 | 决定利润分配的关键 |
| 纵向整合趋势 | 如果龙头企业开始收购上下游，说明格局在变化 |

> 💡 **核心原则：** 一个产业链上，靠近资源端或靠近用户端的环节利润最稳定；中间环节通常最辛苦。

## 三、市场规模：究竟有多大

### 1. 三层市场

| 概念 | 解释 | 案例（证券行业） |
|------|------|-----------------|
| TAM（总可触达市场） | 理论上能做的最大生意 | 中国居民金融资产~300万亿 |
| SAM（可服务市场） | 公司能力范围内能抓到的 | 线上证券交易额~500万亿/年 |
| SOM（可得市场） | 公司实际能拿下的 | 东方财富市占率3.85%×SAM |

### 2. 行业规模怎么查

| 数据源 | 适合什么 |
|--------|---------|
| 行业协会年度报告 | 大多数传统行业 |
| 券商行业深度研报 | 所有行业 |
| 国家统计局 | 宏观相关行业 |
| 上市公司年报"行业分析"部分 | 所有行业（龙头公司年报会写） |
| 咨询公司报告（麦肯锡/BCG/弗若斯特沙利文） | 新兴行业 |

### 3. 增长空间判断

**看渗透率（最常用）：**

| 渗透率 | 含义 | 投资含义 |
|--------|------|---------|
| <5% | 导入期 | 不确定性高，适合风投 |
| 5-30% | 成长期 | 最好的投资阶段，增速快 |
| 30-60% | 成熟早期 | 增速放缓，龙头胜出 |
| >60% | 成熟后期 | 寻找差异化或出海机会 |

**看人均消费 / 国际对标：**

中国大多数行业的人均消费仍低于发达国家，这是长期增长的基础。比如：
- 中国人均证券交易额 vs 美国 → 还有几倍空间（但需要考虑发展阶段差异）

## 四、行业生命周期

### 1. 五个阶段

```
导入期 → 成长期 → 洗牌期 → 成熟期 → 衰退期
  │         │         │         │         │
 多家涌入   高速增长   优胜劣汰   格局稳定   需求萎缩
 看不清谁赢 龙头初现   大量退出   利润稳定   寡头竞争
```

### 2. 各阶段的财务特征

| 阶段 | 营收增速 | 利润率 | 现金流 | 投资重点 |
|------|---------|--------|--------|---------|
| 导入期 | >50%但基数低 | 负或极低 | 巨额融资 | 选对赛道 |
| 成长期 | 20-50% | 提升中 | 可能仍为负 | 看市占率提升 |
| 洗牌期 | 10-20% | 分化明显 | 开始转正 | 选龙头 |
| 成熟期 | 0-10% | 稳定 | 充沛 | 看分红+估值 |
| 衰退期 | 负增长 | 压缩 | 萎缩 | 不投 |

### 3. 判断工具：增速-利润四象限

```
                  营收增速
                   高
               ┌──────┬──────┐
          高   │  明星   │  问题   │
               │（成长期）│（导入期）│
    利润率     ├──────┼──────┤
          低   │  现金牛  │  瘦狗   │
               │（成熟期）│（衰退期）│
               └──────┴──────┘
                   低
```

## 五、增长驱动：结构 vs 周期

### 1. 两种增长

| 增长类型 | 来源 | 可持续性 | 案例 |
|---------|------|---------|------|
| 结构性增长 | 技术变革、渗透率提升、政策推动 | 长期（5-10年） | 新能源车替代燃油车 |
| 周期性增长 | 经济复苏、库存回补、价格波动 | 短期（1-3年） | 钢铁涨价 |

### 2. 如何区分

问三个问题：
1. **这个增长是回补还是新增？**（疫情后旅游反弹 vs 线上办公永久化）
2. **渗透率还有多少空间？**（<30%→结构，>60%→周期）
3. **有没有不可逆的技术或政策驱动？**（碳中和→结构，降息→周期）

> 机构最常犯的错误：把周期性增长当成结构性增长，在周期顶点买入"成长股"。

### 3. 驱动力拆解清单

| 驱动力类型 | 具体因素 | 量化指标 |
|-----------|---------|---------|
| 宏观 | GDP增速、利率、通胀 | 实际GDP、10年期国债收益率 |
| 技术 | 新技术突破、替代进程 | 渗透率、成本曲线 |
| 政策 | 补贴、监管、税收 | 政策文件、补贴金额 |
| 人口 | 老龄化、城镇化 | 年龄结构、城镇化率 |
| 社会 | 消费习惯、文化变迁 | 用户调查、行为数据 |
