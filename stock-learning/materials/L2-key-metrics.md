# L2 - 关键财务指标

读完L1的三表框架后，这一章提取出最常用的财务指标，建立"一眼看穿公司"的能力。

---

## 一、盈利能力指标

### 毛利率

> 公式：毛利 ÷ 营收

**为什么重要：** 反映产品的定价权和竞争力。

| 情形 | 含义 |
|------|------|
| 毛利率高且稳定 | 有护城河，竞争对手难以冲击 |
| 毛利率逐年下降 | 竞争加剧，产品在降价或成本在上升 |
| 毛利率远低于同行 | 没有品牌溢价，只能打价格战 |

### 净利率

> 公式：净利润 ÷ 营收

**为什么重要：** 扣除所有费用后，公司最终能留下多少。

- 毛利率高 + 净利率也高 → 费用控制好（如茅台）
- 毛利率高 + 净利率低 → 费用过高（如部分药企研发+销售费吃掉利润）

### ROE（净资产收益率）

> 公式：净利润 ÷ 股东权益（净资产）

**最重要的综合指标。** 衡量每1块钱股东投入能赚回多少钱。

| ROE | 评价 |
|-----|------|
| >20% | 优秀（巴菲特选股标准之一） |
| 15-20% | 良好 |
| 10-15% | 一般 |
| <10% | 较差，不如买理财 |

> ROE是L4杜邦分析的核心，届时会详细拆解。

## 二、成长性指标

### 营收增长率

> （本期营收 - 上期营收） ÷ 上期营收

**为什么重要：** 公司在变大还是停滞？

- >20%：高速增长（常见于成长股）
- 10-20%：稳定增长
- <10%或负增长：成熟期或衰退期

### 净利润增长率

> 同样逻辑，但需要注意——净利润增长必须和营收增长匹配。

```
危险信号：营收增长5%，净利润增长30%
→ 利润增长不是主业带来，可能是靠卖资产、降费用等一次性手段
```

## 三、运营效率指标

### 应收账款周转天数

> 365 ÷ （营收 ÷ 应收账款）

**代表：** 卖出货后平均多少天能收到钱。

| 天数 | 含义 |
|------|------|
| <30天 | 回款极快，对下游话语权强 |
| 30-60天 | 正常 |
| >90天 | 偏慢，需关注坏账风险 |

### 存货周转天数

> 365 ÷ （营业成本 ÷ 存货）

**代表：** 货物从入库到卖出平均需要多少天。

| 行业特征 | 参考 |
|----------|------|
| 快消品/超市 | 少则20-40天，越快越好 |
| 白酒（高端） | 几年甚至更长（越陈越值钱） |
| 房地产 | 极长（从拿地到交房2-3年） |

> 存货周转不是越短越好——茅台存货天数长是因为酒要陈酿；服装行业存货长意味着过季滞销。

## 四、偿债与安全指标

### 资产负债率

> 总负债 ÷ 总资产

已在L1讲过，这里补充：**关键是看有息负债部分。**

- 应付账款、合同负债等无息负债是好事（占用上下游资金）
- 短期借款、长期借款等有息负债才是真正的压力

### 流动比率

> 流动资产 ÷ 流动负债

| 范围 | 含义 |
|------|------|
| >2 | 短期偿债能力非常好 |
| 1-2 | 正常 |
| <1 | 可能存在流动性问题 |

### 利息保障倍数

> 营业利润 ÷ 利息支出

衡量公司赚的钱够还几次利息。**<1意味着利润不够还利息，危机信号。**

---

## 五、现金流质量指标

### 经营现金流 / 净利润

| 比值 | 含义 |
|------|------|
| >1 | 利润100%真金白银，甚至还有预收款（优秀） |
| =1 | 利润与现金匹配（正常） |
| <1 | 利润有"水分"，应收账款或存货占用了资金（警惕） |

### 自由现金流

> 经营现金流 - 资本开支

> 真正可以自由支配的钱。**价值投资最看重的指标。**
> 自由现金流持续为负的公司，本质是在烧钱维持运营。

## 六、综合实战：用指标判断一家公司

### 案例A：好公司的特征

```
毛利率      > 40%       （有定价权）
净利率      > 15%       （盈利质量好）
ROE         > 15%       （资本回报高）
资产负债率  < 50%       （财务稳健）
经营现金流  > 净利润     （现金真实）
营收增长    > 10%       （在成长）
```

### 案例B：需要警惕的特征

```
毛利率逐年下降         → 产品竞争力在减弱
净利率 > 毛利率的一半   → 费用太高
ROE < 10%              → 资本使用效率低
经营现金流 < 净利润的一半 → 利润可能是纸面
营收增长但毛利率下降   → 靠降价换市场
```

## 快速对照表

| 维度 | 指标 | 好公司 | 差公司 |
|------|------|--------|--------|
| 盈利 | ROE | >15% | <10% |
| 盈利 | 毛利率 | 稳定或提升 | 逐年下滑 |
| 成长 | 营收增速 | >10% | 停滞或下降 |
| 安全 | 资产负债率 | 行业中低水平 | 远超同行 |
| 现金 | 经营现金流/净利润 | >1 | <0.5 |
| 运营 | 应收周转 | 行业领先 | 越来越慢 |
