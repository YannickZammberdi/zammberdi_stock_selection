# L5 - 行业分类与特征

不同的行业有不同的财务特征和估值逻辑。用消费行业的指标去套周期股，会犯大错。

---

## 一、四大行业分类

### 1. 周期性行业

**特征：** 业绩随宏观经济周期大幅波动。

**典型：** 钢铁、煤炭、化工、有色金属、水泥、航运

**财务特征：**

| 时期 | 业绩 | PE | 实际情况 |
|------|------|----|---------|
| 经济繁荣 | 利润暴涨 | PE极低（3-8） | 这时候其实是最贵的 |
| 经济萧条 | 亏损或微利 | PE极高或为负 | 这时候可能是最便宜的 |

> **周期股PE的悖论：** PE最低的时候恰恰是最不该买的时候（利润已经到顶）；PE最高（亏损）的时候往往是买入时机。

**投资逻辑：**
- 在行业低谷（亏损、产能出清）时买入
- 在高潮（满产满销、PE=5）时卖出
- 关键看PB和行业供需周期，PE会骗人

### 2. 成长性行业

**特征：** 行业处于高速扩张期，营收和利润持续快速增长。

**典型：** 新能源、半导体、生物医药、云计算、人工智能

**财务特征：**

| 指标 | 特点 |
|------|------|
| 营收增速 | >20%，甚至>50% |
| 净利润 | 可能为负（烧钱扩张） |
| PE | 通常很高（30-100+）或无意义 |
| 经营现金流 | 常为负（投入期） |

**估值方法：** 看PS（市销率）和营收增速 → 用PEG判断估值是否合理
**风险：** 增速不及预期会杀估值（PE从100跌到30很常见）

### 3. 防御性行业

**特征：** 无论经济好坏都稳定需求，业绩波动小。

**典型：** 公用事业（水电燃气）、食品饮料（必需品）、医药、电信

**财务特征：**

| 指标 | 特点 |
|------|------|
| 营收增速 | 低但稳定（5-15%） |
| 毛利率 | 稳定 |
| 经营现金流 | 好且可预测 |
| 股息率 | 通常较高 |

**投资逻辑：**
- 熊市中表现抗跌
- 适合收股息
- PE通常稳定在15-25之间

### 4. 价值型/成熟行业

**特征：** 行业增速放缓，但龙头公司地位稳固，有稳定现金流。

**典型：** 银行、保险、家电、汽车、房地产

**财务特征：**

| 指标 | 特点 |
|------|------|
| 营收增速 | 低（0-10%） |
| ROE | 中等稳定 |
| 分红 | 高分红率 |

**估值方法：** 看PB、股息率
**投资逻辑：** 赚估值修复+股息的钱

## 二、行业集中度

行业集中度（CR3/CR5）决定了竞争格局：

| 格局 | 特征 | 有利还是不利 |
|------|------|-------------|
| 高度集中（CR3>70%） | 少数巨头垄断，如白酒、空调 | 对龙头有利，定价权强 |
| 分散 | 大量小企业，如餐饮、服装 | 竞争激烈，毛利难提升 |
| 正在集中 | 行业整合期（如啤酒、建材） | 龙头市占率提升期，投资好时机 |

---

## 三、进入壁垒

| 壁垒类型 | 例子 | 是否牢固 |
|----------|------|---------|
| 品牌壁垒 | 茅台、LV | 极强，积累数十年 |
| 技术壁垒 | 芯片设计、创新药 | 强，但可能被新技术颠覆 |
| 牌照壁垒 | 银行、电信、矿山 | 极强，政策许可 |
| 规模壁垒 | 钢铁、面板 | 中等，需要巨额资本投入 |
| 网络效应 | 微信、淘宝 | 极强，越多人用越好用 |
| 渠道壁垒 | 伊利、农夫山泉 | 强，线下铺货需要时间积累 |

**判断行业是否值得投的第一步：问自己，这个行业的赚钱难度大不大？壁垒高不高？**

## 四、不同行业的估值方法对照

| 行业类型 | 首选估值 | 辅助指标 | 关键跟踪 |
|----------|---------|---------|---------|
| 成长型（科技/医药） | PS / PEG | PE（盈利后） | 营收增速、毛利率趋势 |
| 周期型（钢铁/化工） | PB（周期性底部） | PE（顶部参考） | 产能利用率、产品价格 |
| 防御型（公用事业） | PE / 股息率 | EV/EBITDA | 经营现金流稳定性 |
| 金融（银行/保险） | PB | PE | 不良率、拨备覆盖率 |
| 消费（白酒/家电） | PE | 毛利率、ROE | 品牌力、库存 |
| 地产 | NAV/净资产折扣 | PB | 销售回款、负债率 |

## 五、综合判断框架

看一个行业时，按顺序问自己：

```
1. 这是什么类型的行业？（周期/成长/防御/价值）
   → 决定用什么估值方法

2. 行业集中度如何？
   → 龙头是否有定价权

3. 进入壁垒高不高？
   → 现有公司的护城河能否守住

4. 行业处于什么阶段？（导入/成长/成熟/衰退）
   → 决定成长性和投资策略

5. 当前行业的估值在历史什么分位？
   → 贵还是便宜
```

### 实战案例：光伏行业

```
行业类型：成长型→正在向周期型过渡（产能过剩）
集中度：正在集中中，龙头优势明显
壁垒：技术+规模，但技术迭代快
阶段：成长期后半段，产能开始过剩
估值：PE从高点100+回落到15-20

结论：行业增速放缓，但龙头可能通过出清小企业获利
→ 适合用PB+关注产能出清节奏
```

## 六、同行对比分析

仅看单个公司的指标没有意义，必须放在同行中才能判断优劣。选取2-3家可比同行，做横向对比：

| 指标 | 本公司 | 同行A | 同行B | 行业中位数 | 评价 |
|------|--------|-------|-------|-----------|------|
| ROE | | | | | 高于/低于同行？ |
| 毛利率 | | | | | 产品是否有定价优势？ |
| 净利率 | | | | | 费用控制是否优于同行？ |
| 负债率 | | | | | 财务风险是否可控？ |
| 营收增速 | | | | | 增长是否跑赢行业？ |
| PB | | | | | 市场给了多少净资产溢价？ |
| PE | | | | | 估值在同行中偏高还是偏低？ |

**核心要点：**
- **ROE高低差异拆解** — 是利润率更高？周转更快？还是杠杆更高？杜邦分解找原因
- **毛利率是护城河的直接体现** — 如果毛利率显著高于同行，大概率有定价权
- **负债率是安全边际的底线** — 行业低谷期，低负债者活到最后
- **同行对比的价值不在于排名，而在于找差异、问原因**
