# L11 - 个股估值定价（实战版）

> L3教你**认识PE/PB**，L11教你**用它们判断买不买**。

---

## 一、核心原则

一个公司值多少钱，不是看它"好不好"，而是看**它现在多少钱**。

```
好公司 + 贵价格 = 糟糕的投资
普通公司 + 便宜价格 = 不错的投资
好公司 + 便宜价格 = 最佳机会（戴维斯双击）
差公司 + 任何价格 = 不碰
```

---

## 二、不同类型用不同估值指标

| L5分类 | 首选指标 | 次选指标 | 为什么 |
|--------|---------|---------|--------|
| 纯周期（矿业/化工） | **PB** | PS | 利润波动大，PE会骗人 |
| 混合周期（万华/新和成） | **PE + PB** | — | 利润有周期但长期向上 |
| 成长型（科技/医药） | **PEG**（PE÷增速） | PS | 高增速合理给高PE |
| 防御型（超市/公用） | **股息率** | PE | 买它就是为了收息 |
| 价值型（银行/保险） | **PB** | 股息率 | 资产质量和分红更重要 |

## 三、历史分位法（最实用的工具）

不要问"15倍PE贵不贵"，要问"它历史上15倍PE的时候后来怎么样了"。

### 怎么算分位

```
把过去5-10年的PE/PB数据从低到高排
看当前位置排在第百分之几

< 20%分位 → 历史低位区间（值得关注）
20-40%分位 → 偏低
40-60%分位 → 中位（合理）
60-80%分位 → 偏高
> 80%分位 → 历史高位区间（警惕）
```

### 个例：不可能三角

```
PE/PB分位 < 20%（估值低）
   + ROE > 15%（盈利好）
   + 营收在增长（有成长）
   = ⭐ 黄金机会（三者同时出现极罕见）

如果PE低但ROE也在下降（低估值陷阱）→ 不买
如果PE低但营收负增长（价值陷阱）   → 不买
```

## 四、混合周期股的估值实操（以新和成为例）

混合周期 = 利润有波动 + 长期向上。判断标准：

### 买入区
- PE < 历史30%分位
- PB < 历史40%分位
- ROE > 12%（证明利润波动≠盈利能力下降）
- 营收在增长（不是在萎缩）

### 持有区
- PE 30-60%分位
- 利润增速正常
- 什么都不用做

### 卖出区（减仓/清仓）
- PE > 历史70%分位
- 或者 PB > 历史75%分位
- 利润增速明显放缓（营收增速 < 5%）

### 新和成当前判断（2026-06）

| 指标 | 当前值 | 历史区间 | 分位 | 判断 |
|------|--------|---------|------|------|
| PE(TTM) | ~13x | 10-30x | ~25%分位 | ✅ 偏低 |
| PB | ~2.5x | 1.5-5x | ~35%分位 | ✅ 中低 |
| ROE | ~20% | 8-25% | 高分位 | ✅ 优秀 |
| 营收增速 | +15.7% | — | — | ✅ 增长中 |

**结论：处于买入区偏低端**

## 五、快速参考卡

```
┌────────────────────────────────────────────┐
│          个股估值速查（混合周期）            │
├────────────────────────────────────────────┤
│ PE分位 < 20%  → ⭐ 强烈关注                │
│ PE分位 20-40% → ✅ 可以考虑买入             │
│ PE分位 40-60% → ⏸ 持有/观望               │
│ PE分位 60-80% → ⚠ 偏高，考虑减仓           │
│ PE分位 > 80%  → ❌ 卖出或回避               │
│                                            │
│ PB分位 配合PE一起看：                       │
│ PE低位 + PB低位 = 最佳买入窗口              │
│ PE低位 + PB高位 = 资产有泡沫               │
│ PE高位 + PB低位 = 利润低谷期               │
│ PE高位 + PB高位 = 双高，警惕               │
└────────────────────────────────────────────┘
```
