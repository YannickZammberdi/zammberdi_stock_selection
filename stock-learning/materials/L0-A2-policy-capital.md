# L0-A2 - 政策与资金面

## 为什么 L0-A2 比 L0-A1 更重要

A股的底色是 **政策市**。经济好的时候不一定涨（可能收紧），经济差的时候不一定跌（可能放水）。

**L0-A1 告诉你客观的经济位置，L0-A2 告诉你政策对此做出了什么反应，以及市场接没接住。**

> L0-A1 + L0-A2 合在一起，才能判断"现在是 bottom-up 选股的好时机，还是该休息"。

---

## 一、货币政策：水龙头

中国的货币政策由央行（中国人民银行）执行，核心工具：

### 1. 基准利率（LPR）

| LPR变化 | 含义 | 对股市的影响 |
|---------|------|------------|
| 降息（LPR↓） | 央行在放松，鼓励借钱 | 利好，尤其是成长股和地产 |
| 加息（LPR↑） | 央行在收紧，抑制过热 | 利空，尤其是高负债行业 |
| 连续3个月不变 | 政策处于观察期 | 中性 |

**正常范围：**
- 1年期LPR：3.0~4.0%
- 5年期LPR：3.5~4.5%

> 降息周期往往带来估值扩张（水涨船高）。**2024~2026年中国的LPR一直在下行通道。** 这本身就是一个做多信号，只是市场信心不足还在犹豫。

**怎么看趋势：**
- 每次降息后，观察市场是否买账（是否放量上涨）
- 如果是"降息后高开低走"→市场信心极度脆弱
- 如果是"降息后放量上涨"→市场开始配合

### 2. 存款准备金率（RRR）

降准释放银行的放贷能力。一次降准通常释放5000亿~1万亿资金。

| 降准幅度 | 含义 |
|---------|------|
| 0.5% | 常规操作，释放流动性约5000~8000亿 |
| 1.0% | 强力宽松信号 |
| 不降反升 | 收紧信号（近年极少见） |

### 3. 公开市场操作（OMO/MLF）

央行每天通过逆回购/MLF向市场注入或收回短期资金。**关注利率变化而非操作量。**

- MLF利率下降 → 引导LPR下降 → 降息确认
- MLF利率不变 → 政策利率稳定

**一句话判断货币政策取向：**

```
正在连续降息/降准 = 宽松（利好）
刚刚停止降息 = 观察期（中性）
开始加息/升准 = 收紧（利空）
```

## 二、财政政策：政府的钱袋子

财政部决定政府花多少钱、往哪花。

### 核心关注点

| 指标 | 怎么看 | 含义 |
|------|--------|------|
| 财政赤字率 | >3% 积极，<3% 稳健 | 目标赤字率越高，政府花钱越积极 |
| 专项债发行 | 发行速度 vs 计划 | 发行快→基建投资在落地 |
| 特别国债 | 是否发行、用于何处 | 特殊时期的强力刺激手段 |
| 减税降费 | 力度大小 | 减税→企业利润改善 |

### 判断方法

```
赤字率 ↑ + 专项债加速 + 减税 = 积极财政（利好）
赤字率 ↓ + 专项债放缓 = 紧缩财政（利空，近年少见）
```

## 三、产业政策：钱往哪流

> 如果说宏观政策决定的是"要不要进场"，产业政策决定的是"往哪个方向买"。

**A股历史大主线全部由产业政策驱动：**
- 2019~2020：半导体国产替代（国家大基金）
- 2020~2022：新能源（双碳目标）
- 2023~2024：AI（数字经济/新质生产力）
- 2025~2026：？

### 如何跟踪产业政策信号

| 来源 | 频率 | 关注什么 |
|------|------|---------|
| 两会政府工作报告 | 每年3月 | GDP目标、财政赤字率、重点产业方向 |
| 中央经济工作会议 | 每年12月 | 下一年的政策定调 |
| 政治局会议（4/7/10/12月） | 季度 | 经济形势判断+下一阶段政策重点 |
| 国务院常务会议 | 高频 | 具体产业政策出台 |
| 各部委（发改委/工信部） | 不定期 | 行业专项政策 |

**月频更新的做法：**
```
每月20号，打开"中国政府网"或财经App，搜"政策"看当月有哪些重要文件，
记录1~2条最关键的政策方向。
```

## 四、资金面：水来了没有

政策再好，资金不跟进也是白搭。

### 1. 北向资金（沪深港通）

**代表外资态度。** 对A股短期走势有风向标意义。

| 北向行为 | 含义 |
|---------|------|
| 连续5天净流入 | 外资看好A股 |
| 连续5天净流出 | 外资在撤退 |
| 单日 >100亿净流入 | 强烈看多信号 |
| 单日 >100亿净流出 | 强烈看空信号 |

**月度累计：** 月净流入>300亿 = 外资非常积极；月净流出>300亿 = 外资在系统性地撤。

> 北向资金不是永远对，但它是一个**先行指标**。北向连续买入时A股短期不太会跌。

### 2. 两融余额（融资融券）

**代表散户和游资的杠杆水平。**

| 两融余额 | 含义 |
|---------|------|
| 持续上升 | 市场情绪乐观，资金在加杠杆 |
| 持续下降 | 市场情绪悲观，在去杠杆 |
| 到历史高位（如1.6万亿+） | 过热信号，风险增大 |
| 到历史低位（如1.2万亿-） | 极度悲观，可能见底 |

### 3. 新发基金规模

**代表散户的钱在不在进场。**

| 情况 | 含义 |
|------|------|
| 月度新发 > 1000亿 | 市场很热，资金在涌入 |
| 月度新发 < 300亿 | 市场冷淡，没人买基金 |
| 出现"日光基"（一天卖完） | 顶部信号（散户情绪极亢奋） |
| 基金发行冰封 | 底部信号（散户情绪极悲观） |

### 4. 股权风险溢价（ERP）

**比较专业的指标，但非常有用。**

```
ERP = 1/市盈率 - 十年期国债收益率
    = 买股票能多赚多少相对于买债券
```

| ERP | 含义 |
|-----|------|
| 历史高位（>5%） | 股票相对于债券极度便宜 → 底部区域 |
| 历史低位（<2%） | 股票相对于债券极度贵 → 顶部区域 |

> 具体数字随市场变化，但逻辑不变：**ERP高时买、低时卖。** 历史上A股的大底都是ERP最高的时候。

## 五、综合判断：政策是否友好

### 政策+资金面评分表

| 项目 | 宽松/流入（+1分） | 中性（0分） | 收紧/流出（-1分） |
|------|-----------------|------------|-----------------|
| 货币政策 | 降息/降准中 | 利率不变 | 已开始加息 |
| 财政政策 | 赤字率>3%+专项债加速 | 正常 | 紧缩 |
| 产业政策 | 有明显的大方向（如AI/新能源） | 没有明确主线 | 压降某个行业 |
| 北向资金 | 月净流入>200亿 | 小幅波动 | 月净流出>200亿 |
| 新发基金 | >800亿/月 | 300~800亿 | <300亿 |
| ERP | 历史高分位（>5%） | 中间 | 历史低分位（<2%） |

```
总分 ≥ +3：全力做多（环境极度友好）
总分 0~+2：中性偏多（正常操作）
总分 -2~-1：中性偏空（谨慎）
总分 ≤ -3：休息或轻仓（环境极差）
```
