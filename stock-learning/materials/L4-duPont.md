# L4 - 杜邦分析拆解

杜邦分析只有一个目标：**把ROE拆开，看一家公司到底靠什么赚钱。**

---

## 一、ROE为什么重要

**ROE（净资产收益率）是所有指标的终点。** 巴菲特说："我选择的公司，ROE要超过20%且持续多年。"

但只看ROE数字不够。两家公司都是ROE=20%，赚钱方式可能完全不同：

- 公司A：高利润率 × 低杠杆（安全）
- 公司B：低利润率 × 高杠杆（危险）

杜邦分析就是把ROE拆成三部分来看。

---

## 二、杜邦拆解公式

```
ROE = 净利润率 × 资产周转率 × 权益乘数
```

拆开看：

| 因子 | 公式 | 反映什么 |
|------|------|----------|
| **净利润率** | 净利润 ÷ 营收 | 卖100块能赚多少？（盈利能力） |
| **资产周转率** | 营收 ÷ 总资产 | 1块钱资产能做多少生意？（运营效率） |
| **权益乘数** | 总资产 ÷ 净资产 | 用了多少杠杆？（财务风险） |

### 三种赚钱模式

**模式一：高利润型（高净利润率）**

```
典型特征：净利润率高，周转率低，杠杆低
代表公司：茅台、恒瑞医药、海天味业

ROE = 高净利润率 × 低周转 × 低杠杆
```

> 核心是"卖得贵"——产品有定价权，品牌溢价强。
> 这是最优质的商业模式，可持续性强，风险低。

**模式二：高周转型（高周转率）**

```
典型特征：净利润率低，周转率高，杠杆中等
代表公司：沃尔玛、永辉超市、小米

ROE = 低净利润率 × 高周转 × 中等杠杆
```

> 核心是"卖得快"——单品赚得少，但一年周转很多次。
> 考验的是供应链管理能力，做好了一样很赚钱。

**模式三：高杠杆型（高权益乘数）**

```
典型特征：净利润率中等，周转率中等，杠杆极高
代表公司：银行、房地产

ROE = 中等利润率 × 中周转 × 高杠杆
```

> 核心是"借鸡生蛋"——用别人的钱放大了自己的回报。
> 经济好时ROE很高，经济差时杠杆会反噬（亏钱也加倍）。

## 三、实战拆解：三家公司ROE对比（2023年）

| 分解项 | 贵州茅台 | 沃尔玛 | 某银行 |
|--------|---------|-------|-------|
| ROE | 30% | 15% | 10% |
| 净利润率 | 49% | 2.5% | 25% |
| 资产周转率 | 0.5次 | 2.4次 | 0.05次 |
| 权益乘数 | 1.23 | 2.5 | 12 |

**解读：**

```
茅台：ROE=49%×0.5×1.23=30%
→ 49%的净利润率是绝对核心，极高的定价权

沃尔玛：ROE=2.5%×2.4×2.5=15%
→ 利润率极低，但周转率2.4次（约5个月转一轮），杠杆正常

某银行：ROE=25%×0.05×12=10%
→ 利润率不错，但资产周转极慢（贷款是长期资产）
→ 靠12倍杠杆放大，风险最大
```

## 四、杜邦分析怎么用

### 4.1 判断公司类型

拿到一家公司，先做杜邦拆解：

```
高利润率 + 低杠杆 → 最优质（护城河深）
高周转 + 低杠杆 → 运营能力强
高杠杆 → 需要警惕，看现金流能否覆盖利息
```

### 4.2 找变化原因

当ROE发生变化时，杜邦能告诉你问题出在哪：

```
例：某公司ROE从20%降到12%

拆解发现：
  净利润率：15% → 14%（变化不大）
  资产周转率：1.2 → 0.8（大幅下降！）
  权益乘数：1.1 → 1.1（没变）

结论：问题出在周转率——资产变多了但营收没跟上
→ 可能新投的产能没有产生预期收益
```

### 4.3 交叉验证

杜邦可以帮助发现"异常公司"：

```
如果一家公司ROE很高，但不是靠高利润率：
→ 要么是运营天才（周转率极高）
→ 要么是杠杆太大（隐含风险）

如果ROE高但经营现金流差：
→ 利润可能是纸面的，赚的钱变成了应收账款
```

---

## 五、杜邦分析的局限

| 局限 | 说明 |
|------|------|
| 只看过去 | 基于历史数据，不能预测未来 |
| 忽略现金流 | 高ROE但无现金流的公司很常见 |
| 杠杆风险 | 高杠杆推高的ROE不可持续 |
| 会计操纵 | 净利润可能被美化（如减值推迟） |

**所以杜邦分析必须结合：**
- L2的现金流指标
- L5的行业特征
- 分季度看趋势
