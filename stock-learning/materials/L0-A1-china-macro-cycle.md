# L0-A1 - 中国宏观周期

## 为什么先看宏观？

你分析的个股再好，如果大盘在系统性下跌，它也很难独善其身。**80%的个股走势由大势决定，20%才是公司自身。** 宏观判断回答的是最底层的问题：

> 现在该重仓、轻仓、还是空仓？

---

## 一、美林时钟：理解经济周期的基本框架

### 四个象限

```
               通胀上升（CPI↑）
                    │
            ┌───────┴───────┐
        经济 │   复苏期     │   过热期    经济
        好 │  （高增长+低通胀）│  （高增长+高通胀） 差
         │               │               │
            ├───────────────┼───────────────┤
         │   衰退期     │   滞胀期     │
        差 │  （低增长+低通胀）│  （低增长+高通胀） │
            └───────────────┴───────────────┘
                    │
               通胀下降（CPI↓）
```

| 象限 | 经济 | 通胀 | 最佳资产 |
|------|------|------|---------|
| 复苏 | ↑ | ↓或低 | 股票（成长股最佳） |
| 过热 | ↑ | ↑ | 商品、周期股 |
| 滞胀 | ↓ | ↑ | 现金、防御性资产 |
| 衰退 | ↓ | ↓ | 债券 |

### 拿一个具体年份来感性地理解

| 时段 | 象限 | 当时的感觉 |
|------|------|-----------|
| 2019年 | 复苏 | PMI回升、CPI温和、降息周期 → 股票大年 |
| 2021上半年 | 过热 | PPI飙升、大宗商品暴涨、央行开始收紧 → 商品暴涨 |
| 2022年 | 滞胀 | 疫情封控+通胀+美联储加息 → 股债双杀 |
| 2023下半年~2024 | 弱复苏/通缩 | PMI在荣枯线徘徊、CPI接近0、房地产下行 → 结构性行情 |

> **关键认知：A股不总是按美林时钟走。因为中国有政策干预，经济在衰退时央行放水，股市反而能涨。** 所以A股需要结合L0-A2一起来看。

## 二、判断经济位置的五个核心指标

### 1. PMI（采购经理人指数）

**怎么看数据：** 每月第一个工作日公布。

| 数值 | 含义 |
|------|------|
| >50 且上升 | 经济在扩张 |
| >50 但下降 | 扩张在放缓 |
| <50 且下降 | 经济在收缩 |
| <50 但上升 | 收缩在放缓，可能见底 |

**正常范围：** 48~52是正常波动。**>52算偏热，<48算偏冷。**

> 关注趋势：连续3个月方向一致才算确认。

**怎么查：** 国家统计局官网、东方财富App搜"PMI"。

### 2. 社融（社会融资规模）

**这是A股最关键的宏观指标之一。** 社融代表"全社会借了多少钱"，也就是经济活动的活跃度。

**怎么看数据：** 每月10~15日公布上月数据。

| 社融同比多增 | 含义 |
|-------------|------|
| 多增 > 5000亿 | 宽信用，经济在复苏（利好） |
| 多增 0~5000亿 | 正常 |
| 少增 | 信用收缩（利空） |

**极端值：**
- 连续3个月同比少增 → 经济在加速下行
- 连续3个月同比多增超万亿 → 政策在强力托底

### 3. CPI（居民消费价格指数）

**代表：** 老百姓的物价水平。

| CPI | 含义 |
|-----|------|
| 1~3% | 正常范围 |
| <1% | 通缩风险（消费不振，经济偏冷） |
| >3% | 通胀偏高（可能引发收紧） |
| >5% | 严重通胀（必须警惕） |

**A股2023~2024的特殊情况：** CPI长期在0附近，本质是通缩——消费弱、企业不敢涨价、工资不涨。通缩对股市的影响是：名义增长低，企业盈利弱。

### 4. PPI（工业生产者出厂价格指数）

**代表：** 上游工业品的价格。和周期股盈利直接相关。

| PPI | 含义 |
|-----|------|
| >0 且上升 | 上游涨价，周期股盈利好 |
| >0 但下降 | 涨价在放缓 |
| <0（负值） | 上游通缩，原材料企业利润受压 |

**正常范围：** -3%~3%。

> PPI与周期股（化工、钢铁、有色）的盈利高度同步。PPI见底回升时，往往是周期股的最佳买点。

### 5. 十年期国债收益率

**代表：** 市场对经济和通胀的预期。

| 收益率 | 含义 |
|--------|------|
| 2.5~2.8% | 极低，市场预期经济很弱（债牛股可能也牛，但偏防御） |
| 2.8~3.2% | 正常范围 |
| >3.2% | 经济预期好转，成长股承压（估值受压制） |
| >3.5% | 过热信号，警惕收紧 |

> 十年期国债收益率下降 = 债市上涨（利率下行）。**历史上A股大底往往对应国债收益率极低的时候。**

## 三、综合象限判断表

把五个指标放在一起看，判断当前经济在哪个位置：

| 经济状态 | PMI | 社融 | CPI | PPI | 国债收益率 |
|---------|-----|------|-----|-----|----------|
| 复苏 | >50趋势向上 | 同比多增 | 1~2% | 从负转正 | 2.8%左右企稳 |
| 过热 | >52 | 多增但开始放缓 | >3% | >3% | >3.2%且上升 |
| 滞胀 | 回落趋势 | 少增 | >3% | 高位波动 | 高位 |
| 衰退 | <50 | 连续少增 | <1% | 负值 | 2.5~2.8% |
| 弱复苏（当前常见） | 50附近波动 | 一般或略多 | 0~1% | 负值或0附近 | 2.5~2.8% |

## 四、对投资的含义

| 经济状态 | 怎么做 |
|---------|-------|
| 复苏 | 重仓，优先配成长股（科技、消费） |
| 过热 | 减仓，转配周期股（有色、化工、煤炭） |
| 滞胀 | 轻仓，只配防御（公用事业、红利、现金） |
| 衰退 | 开始布局，债券为主，股票分批建仓 |

**但注意：** 当政策强力逆周期发力时（如2024年央行降准降息+国家队进场），衰退期也可以做多——经济差但"政策底"已现。这就是L0-A2的作用。
