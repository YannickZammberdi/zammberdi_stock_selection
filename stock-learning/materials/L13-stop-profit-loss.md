# L13 - 止盈止损

> L11告诉你**值多少**，L12告诉你**买多少**，L13告诉你**什么时候走**。

---

## 一、核心原则

赚钱不卖等于没赚，亏钱不卖等于锁定亏损。

```
买入前先想好：什么情况我会走？
然后把规则写下来，到了就执行——不犹豫、不找借口。
```

---

## 二、止损：先活下来

### 两类止损（必须同时设）

| 类型 | 含义 | 执行方式 |
|------|------|---------|
| **硬止损** | 股价跌到某个价格就走 | 机械执行，不找理由 |
| **基本面止损** | 买入逻辑被证伪就走 | 不看价格，逻辑变了就该走 |

### 硬止损设置

| 股票类型 | 止损幅度 | 理由 |
|---------|---------|------|
| 纯周期（矿业） | -20% | 波动大，设窄了容易被震出去 |
| 混合周期（化工/制造） | -15% | 波动中等 |
| 成长型（科技） | -20~25% | 高波动高回报，宽一点 |
| 防御型（收息） | -10% | 买它就是为了稳，跌多了说明逻辑变了 |

### 基本面止损清单（出现任意一条就走）

```
□ 核心产品价格崩盘（如新和成维生素价格跌30%+）
□ ROE连续两季下降超过5个百分点
□ 负债率突然跳升>10个百分点
□ 营收增速转负
□ 核心管理层离职/创始人减持
□ 财报被出具非标意见/被监管立案
```

---

## 三、止盈：知道赚什么钱

不同股票，止盈逻辑不一样。先回答L10最后一个问题：**这笔交易我赚什么钱？**

### 三种回报来源的止盈

| 回报来源 | 止盈方式 | 典型幅度 | 持有时间 |
|---------|---------|---------|---------|
| **估值修复** | 估值回到合理区间就卖 | 30-100% | 1-2年 |
| **业绩增长** | 等增速放缓/估值过高才卖 | 每年15-30% | 3-5年 |
| **股息收入** | 只要股息率ok就不卖 | 每年3-7% | 长期 |

### 按L5分类的止盈规则

| 类型 | 止盈信号 |
|------|---------|
| **纯周期** | PB > 历史60%分位 + 满产满销 → 卖出 |
| **混合周期** | PE > 历史70%分位 + 或营收增速 < 5% → 减仓/清仓 |
| **成长型** | PS > 20x + 营收增速下台阶 → 卖出 |
| **防御型** | 股息率 < 历史20%分位 → 换更高股息标的 |
| **价值型** | PB > 历史60%分位 → 卖出 |

### 移动止盈（适用于涨了以后）

```
股价从买入价涨了30%以上 → 把止损位上移到成本价（保本）
股价从买入价涨了50%以上 → 把止损位上移到浮盈的50%位置

例：
10元买入 → 涨到13元 → 止损位上移到10元（至少不亏）
           → 涨到15元 → 止损位上移到12.5元（保留一半利润）
           → 涨到20元 → 止损位上移到15元（保留大部分利润）
```

## 四、执行纪律

### 止损不需要"确认"

```
股价触发止损价 → 当天卖出（不等收盘、不等反弹）

"再看看"是止损最大的敌人。
大多数散户的巨亏都来自：该止损时没止，然后说服自己"拿着吧反正长期"。
```

### 止盈可以分批

```
估值进入偏高区间 → 先卖一半
估值进入危险区间 → 全卖
移动止盈位上移后触发 → 全卖
```

### 卖出后不要马上买回来

```
卖了之后 → 至少等1个月再考虑是否重新买入

为什么？
- 刚卖就涨会冲动追回（情绪交易）
- 需要时间重新评估基本面和估值
- 空仓也是策略，不是一定要满仓
```
