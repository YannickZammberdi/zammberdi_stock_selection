# L10 - 周期、政策与时机

## 用途

L10回答最后一个问题：**现在该不该买？** L8告诉你行业长什么样，L9告诉你谁在赚钱，L10告诉你什么时候出手。

---

## 一、周期的类型与识别

### 1. 三种周期

| 周期类型 | 时间跨度 | 驱动因素 | 识别方法 | 典型行业 |
|---------|---------|---------|---------|---------|
| 库存周期（基钦周期） | 3-4年 | 企业补库/去库 | 库存水平、PMI | 制造业、大宗商品 |
| 资本开支周期（朱格拉周期） | 7-10年 | 设备更新/产能建设 | 产能利用率、资本开支增速 | 矿业、航运、化工 |
| 超级周期（长期） | 20-30年 | 技术革命/人口/城市化 | 渗透率、结构性变化 | 所有行业 |

### 2. 库存周期的四个阶段

```
              需求上升
                │
           ┌────┴────┐
       库存  │  复苏期   │  繁荣期   库存
       低   │（主动补库）│（被动去库）  高
           ├─────────┼─────────┤
           │  衰退期   │  萧条期    │
           │（被动补库）│（主动去库）  │
           └─────────┴─────────┘
                │
              需求下降
```

| 阶段 | 需求 | 库存 | 价格 | 最佳策略 |
|------|------|------|------|---------|
| 复苏期 | ↑ 回升 | ↓ 低 | 开始涨 | ✅ 买入（最佳时机） |
| 繁荣期 | ↑ 高 | ↑ 堆积 | 高位震荡 | ⚠ 逐步卖出 |
| 衰退期 | ↓ 回落 | ↑ 过剩 | 开始跌 | ❌ 卖出 |
| 萧条期 | ↓ 低 | ↓ 出清 | 跌到低谷 | ✅ 开始关注（准备买入） |

> 大多数人做反了：繁荣期看到满产满销+PE很低时买入，萧条期看到亏损+PE极高时割肉。**周期股在亏损时关注、盈利暴涨时卖出。**

### 3. 判断周期位置的三把尺

**第一把尺：产能利用率**

| 数值 | 位置 | 含义 |
|------|------|------|
| <65% | 底部 | 行业大面积亏损，产能出清中 |
| 65-80% | 复苏 | 需求恢复，龙头企业开始赚钱 |
| 80-90% | 繁荣 | 供需偏紧，利润丰厚 |
| >90% | 过热 | 接近满产，暴利但接近顶部 |

**第二把尺：库存水平**

用"库存/销售比"来判断：该比值处在历史低位→补库刚开始；处在历史高位→去库刚开始。

**第三把尺：价格分位**

| 价格分位 | 位置 | 判断 |
|---------|------|------|
| 历史底部的10-20% | 底部区间 | ✅ 值得关注 |
| 中位数附近 | 合理 | ⏸ 观望或小仓位 |
| 历史顶部的10-20% | 过热区间 | ❌ 应卖出 |

## 二、PE悖论：为什么周期股PE会骗人

### 1. PE悖论图示

```
                               利润曲线
                              ╱    ╲
                            ╱        ╲
                          ╱            ╲
                        ╱                ╲
                      ╱                    ╲
                    ╱                        ╲
                  ╱                            ╲
             PE极低（股价在顶部）          PE极高（股价在底部）
                ↑陷阱                         ↑机会
```

**规律：**
- PE最低时（利润最好）→ 股价通常在顶部
- PE最高时（利润最差）→ 股价通常在底部
- PE为负时（亏损）→ 有些是深坑（价值陷阱），有些是机会（周期底部）

### 2. 如何避免被PE骗

| 公司类型 | 该看哪个指标 | 原因 |
|---------|------------|------|
| 纯周期公司 | PB（市净率） | PE在周期顶部极低、底部极高，会误导 |
| 成长型公司 | PE可参考，配合PEG | 利润持续增长，PE有参考意义 |
| 金融公司 | PB为主，PE为辅 | 资产质量比利润更可靠 |
| 亏损公司 | PS或看现金消耗 | PE为负时没有意义 |

### 3. PB估值的用法

PB适用于：**周期性行业、金融行业、资产密集型行业**

| PB | 含义 |
|----|------|
| <1.0x | 破净——市场认为公司资产质量有问题（可能是机会也可能是陷阱） |
| 1.0-1.5x | 偏低——如果资产质量没问题，可能被低估 |
| 1.5-3.0x | 合理偏高——取决于ROE水平 |
| >3.0x | 溢价——市场给了品牌/技术/成长溢价 |

**PB结合ROE判断：**

```
P/B = (ROE - g) / (r - g)

简化理解：PB是市场愿意为每1元净资产付多少钱
ROE越高 → 合理的PB应该越高

参考标准：
ROE 5%  → 合理PB 0.5-0.8x（低于1说明回报率不如债券）
ROE 10% → 合理PB 1.0-1.5x
ROE 15% → 合理PB 1.5-2.5x
ROE 20%+ → 合理PB 2.5-5.0x（茅台这种可以更高）
```

## 三、政策分析框架

### 1. 三类政策影响

| 政策类型 | 影响方式 | 时间跨度 | 示例 |
|---------|---------|---------|------|
| 宏观政策 | 利率、流动性、财政 | 1-3年 | 降息利好券商、地产 |
| 行业政策 | 补贴、准入、监管 | 3-5年 | 光伏补贴退坡、证券佣金率放开 |
| 技术标准 | 强制/推荐标准 | 5-10年 | 国六排放标准、钙钛矿标准 |

### 2. 政策分析三步法

**第一步：识别政策信号**
- 国务院/部委文件 → 有约束力
- 行业协会倡议 → 指导性，约束力弱
- 领导讲话 → 方向性，需后续文件落实

**第二步：判断影响方向**
- 利好还是利空？
- 一次性冲击还是持续性影响？
- 对行业龙头是否更有利？（监管趋严通常利好龙头）

**第三步：估算影响幅度**
- 量化测算（如果可以）：补贴退坡→利润减少X%
- 定性判断：影响很大/中等/可忽略

### 3. 政策周期与市场周期的关系

```
政策宽松期 ─→ 行业复苏 ─→ 政策收紧
     ↑                          ↓
     ←─── 行业萧条 ←─── 政策宽松期
```

> 留意：政策通常是逆周期的——行业过热时收紧，行业低迷时放松。在政策放松初期买入，收紧初期卖出。

### 4. 政策与时机速查

| 政策类型 | 对投资时机的含义 |
|---------|----------------|
| 降息/降准 | 利好金融、地产、高负债行业 |
| 加息/紧缩 | 利好现金充裕的公司，利空高杠杆行业 |
| 补贴退坡 | 短期利空（市场先跌再说），长期利好龙头（弱者出局） |
| 监管加强 | 对龙头利好（合规成本对小公司更致命） |
| 牌照放开 | 利空原有持牌公司（竞争加剧） |
| 国企改革/重组 | 个案分析，通常短期炒作 |

## 四、买入清单

### 1. 通用买入检查清单

在买入任何一只股票前，逐条检查：

```
□ L8 行业画像
  □ 行业定义清楚，知道它做什么
  □ 产业链利润环节清晰
  □ 市场规模有数据支撑
  □ 增长驱动力已识别（结构/周期/混合）

□ L9 竞争格局
  □ 知道行业当前格局类型
  □ CR3趋势是升还是降
  □ 龙头有可持续的壁垒
  □ 驱动力公式已列出

□ L10 时机判断
  □ 库存周期位置已判断（当前处于哪个阶段）
  □ PB/PE在历史偏低分位（<30%）
  □ 行业不是政策收紧期
  □ 明确知道这笔交易赚的是什么钱（估值修复/业绩增长/股息）
```

### 2. 按行业类型的买入信号

| 行业类型 | 买入信号 | 卖出信号 |
|---------|---------|---------|
| 纯周期 | PB<历史20%分位 + 行业亏损/停产 + 产能出清中 | PB>历史60%分位 + 满产满销 |
| 混合周期 | PE<历史30%分位 + 行业有增长 + PB合理 | PE>历史70%分位 + 增速放缓 |
| 成长型 | PS<10x + 营收增速>30% + 市占率提升 | PS>20x + 增速下台阶 |
| 防御型 | 股息率>历史80%分位 + CFO能覆盖分红 | 股息率<历史20%分位 |
| 价值型 | PB<历史20%分位 + ROE稳定 | PB>历史60%分位 |

## 五、预期回报来源：清楚知道自己赚什么钱

### 1. 三种回报来源

| 回报来源 | 确定性 | 幅度 | 时间 | 典型操作 |
|---------|-------|------|------|---------|
| 估值修复 | 中 | 30-100% | 1-2年 | PB<1买入，PB>1.5卖出 |
| 业绩增长 | 低-中 | 每年15-30% | 3-5年 | 成长股，持有穿越周期 |
| 股息收入 | 高 | 每年3-7% | 长期 | 防御股，分红再投资 |

### 2. 交易计划模板

每次买入前写下来：

```
买入日期：________  价格：________
行业：________  股票：________

买入逻辑：
1. 赚什么钱？ □ 估值修复 □ 业绩增长 □ 股息
2. 为什么现在买？（触发信号）________________
3. 预期回报：____%  预期时间：____个月

卖出条件：
1. 止盈：股价达到____（+__%）或 PE达到____
2. 止损：股价跌破____（-__%）或 基本面恶化迹象：________
3. 时间止损：____个月后未达预期 → 重新评估

跟踪指标（触发卖出就盯这些，不看别的）：
1. ________  2. ________  3. ________
```

> 💡 **写完计划再下单。** 大多数亏损都来自"先买了再说"。如果你写不出"赚什么钱"和"为什么现在"，说明还没想清楚。

## 七、L8-L10与L7的衔接

```
L8 行业画像     → 你知道这是什么行业，产业链怎么分工
L9 竞争驱动力   → 你知道谁在赚钱，利润靠什么驱动
L10 周期时机   → 你知道现在处于什么位置

↓

L7 量化初筛     → 用L8-L10的认知调整筛子阈值：
                  - 周期性行业→跳过毛利率，用PB
                  - 轻资产成长→关注营收增速和PS
                  - 防御型行业→关注股息率和现金流

↓

L1-L6 个股分析  → 对候选池逐一做深度财务分析

↓

L6 综合判断     → 结合行业理解（L8-L10）+ 行业筛选（L7）
                  + 个股分析（L1-L5）= 最终决策
```

**一个完整的投资决策需要完整的链条：** 懂行业 → 懂筛选 → 懂个股。跳过任何一环，决策质量都会下降。

---

> 💡 L10是L8-L9的终点，也是投资决策的起点。学完L10，你应该能回答三个问题：
> 1. 现在该不该买这个行业？（周期位置+政策方向）
> 2. 买谁？（L9的竞争格局结论）
> 3. 什么时候走？（买入时就想好卖出条件）
