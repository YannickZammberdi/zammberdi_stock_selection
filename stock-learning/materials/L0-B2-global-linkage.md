# L0-B2 - 全球联动与ASX情绪

## ASX在全球中的位置

ASX不是孤立的市场。作为开放经济体，它受到全球资金的直接影响。

### 三层影响

```
第一层：美联储利率 → 全球资金成本 + 美元强弱
第二层：中国需求 → 大宗商品价格 + ASX资源股
第三层：全球避险情绪 → 金价 + 医疗股
```

---

## 一、美联储利率：全球资产定价的锚

| 美联储状态 | 对ASX的影响 |
|-----------|------------|
 | 降息 | 全球资金充裕 → 利好ASX（尤其成长股） |
 | 维持 | 中性，取决于市场预期 |
 | 加息 | 全球资金收紧 → 利空ASX |

> 美联储降息周期往往带动RBA跟随降息，双重利好ASX。

### 怎么看

关注美联储的 **点阵图** 和 **鲍威尔讲话**。关键信息：

- "不急于降息" → 利空（高利率更久）
- "通胀在回归2%" → 利好（降息预期升温）

**月频做法：** 每月留意一次美联储议息会议结果（每年8次，约6周一次），主要看利率不变还是有变化。

---

## 二、美债收益率：全球资金的"引力场"

| 10年期美债收益率 | 对ASX的含义 |
|----------------|------------|
 | <4.0% | 全球资金愿意流向风险资产 → 利好ASX |
 | 4.0~4.5% | 正常范围 |
 | >4.5% | 资金回流美国 → 利空ASX（尤其高估值成长股） |

> **美债收益率是全球股市的"地心引力"。** 收益率越高，股票估值越低。

---

## 三、美元指数（DXY）

| 美元走势 | 对ASX的含义 |
|---------|------------|
 | 美元弱（DXY↓） | 大宗商品涨 → 利好ASX |
 | 美元强（DXY↑） | 大宗商品跌 → 利空ASX |
 | 美元急剧走强 | 新兴市场资金外流 → ASX也会被波及 |

---

## 四、ASX 市场情绪指标

### 不需要太复杂的指标，ASX看这几个就够了：

| 指标 | 怎么看 |
|------|--------|
 | ASX200 技术面 | 是否在200日均线上方？趋势向上还是向下？ |
 | VIX（恐慌指数） | >30 → 全球避险 → ASX资源股跌、金矿股涨 |
 | 澳洲消费者信心 | 连续低下说明国内经济偏弱 |
 | ASX板块轮动 | 资金在银行/矿业/医疗/科技之间怎么流动 |

### 简单的技术判断

```
ASX200 > 200日均线 + 周线趋势向上 = 强势
ASX200 < 200日均线 + 周线趋势向下 = 弱势
ASX200 在200日均线附近震荡 = 方向不明，精选个股为主
```

## 六、整合使用方式

### 分析ASX个股前的宏观检查清单

1. **美联储利率方向？** （降息→利好，加息→利空）← L0-B2
2. **大宗商品趋势？** （铁矿石/金价往哪走）← L0-B1
3. **中国经济怎么样？** （PMI/社融）← 关联L0-A1
4. **ASX200技术面？** （在均线上还是下）← L0-B2

> 以上4点，5分钟扫完。
