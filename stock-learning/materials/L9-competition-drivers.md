# L9 - 竞争格局与驱动力

## 用途

L8告诉你行业长什么样。L9回答：**这个行业谁在赚钱？怎么赚的？凭什么他们能赚？** 核心是三步：竞争格局 → 竞争壁垒 → 驱动力拆解。

---

## 一、竞争格局分析

### 1. 市场结构类型

| 类型 | 公司数 | 产品差异 | 定价权 | 典型行业 | 投资含义 |
|------|--------|---------|-------|---------|---------|
| 完全竞争 | 极多 | 无差异 | 无（价格接受者） | 农产品、大宗商品 | 别碰，没人能持续赚钱 |
| 垄断竞争 | 较多 | 有差异 | 弱 | 证券、服装、餐饮 | 龙头靠品牌/规模赚超额利润 |
| 寡头垄断 | 3-5家 | 有或无异 | 强 | 银行、通信、航空 | 龙头利润稳定，但增速慢 |
| 完全垄断 | 1家 | 无替代 | 极强 | 公用事业、铁路 | 利润稳定，看监管政策 |

> 大多数值得投资的公司都属于**垄断竞争**或**寡头垄断**。完全竞争的行业很难跑出长期牛股。

### 2. 集中度指标

**CR3 / CR5：前3/5家市场份额之和**

| CR3 | 含义 | 投资含义 |
|-----|------|---------|
| <30% | 分散市场 | 有整合机会，但不确定性高 |
| 30-50% | 适度集中 | 竞争激烈，龙头在扩大优势 |
| 50-70% | 高度集中 | 龙头溢价明显，格局稳定 |
| >70% | 寡头市场 | 新进入者很难，龙头护城河深 |

**CR3变化趋势比绝对值更重要：**
- CR3在上升 → 龙头正在吃掉弱者的份额 → 利好龙头
- CR3在下降 → 新进入者在抢地盘 → 行业在恶化

### 3. 集中度计算（快速方法）

如果没有精确数据，可以用近似法：

```
CR3 ≈ 前3上市公司营收之和 ÷ 行业总营收

行业总营收 ≈ 上市公司营收之和 ÷ 上市率

上市率 ≈ 该行业上市公司总市值 ÷ 行业估算总市值
```

> 不需要精确到小数点。知道"CR3大概从40%升到55%了"比知道"CR3=53.7%"更有用。

### 4. 市场格局的四种演化路径

```
                    龙头份额
                     上升
                 ┌────────┬────────┐
            产品  │  赢家通吃  │  百花齐放  │
            差   │  （科技）  │  （服装）   │
            异   ├────────┼────────┤
            化   │  寡头稳定  │  价格战泥潭  │
            小   │  （银行）  │  （钢铁）   │
                 └────────┴────────┘
                     下降
```

| 象限 | 特征 | 投资策略 |
|------|------|---------|
| 赢家通吃 | 网络效应/规模效应，龙头越来越大 | 选龙头，不要选老二 |
| 百花齐放 | 差异化空间大，各有各的饭吃 | 选产品力最强的 |
| 寡头稳定 | 大家都默契不降价，利润稳定 | 选分红最高的 |
| 价格战泥潭 | 只能靠降价抢份额，谁都不赚钱 | 最好不碰 |

### 5. 行业格局判断清单

```
□ 近3年CR3是升还是降？______
□ 行业老大是否有超额利润（ROE比老二高5pct以上）？______
□ 新进入者多吗？（上市<5年的公司占比）______
□ 行业是否有"双龙头"或"一超多强"的稳定结构？______
□ 客户转换成本高吗？（换一家供应商需要重新培训/改造）______

结果判断：
- 4-5个✓ → 格局优秀，适合重仓龙头
- 2-3个✓ → 格局一般，需要个案分析
- 0-1个✓ → 格局差，谨慎投资
```

## 二、竞争壁垒：为什么别人进不来

### 1. 八大壁垒

| 壁垒类型 | 含义 | 典型案例 | 可维持性 |
|---------|------|---------|---------|
| 品牌 | 消费者认牌子 | 茅台、可口可乐 | 很长（除非重大危机） |
| 技术/专利 | 别人做不了或做了侵权 | 高通、华为5G | 有限期（专利过期或被超越） |
| 规模效应 | 规模越大成本越低 | 台积电、亚马逊AWS | 较长（先发优势+资本壁垒） |
| 网络效应 | 用的人越多越好用 | 微信、东方财富 | 很长（用户粘性极强） |
| 转换成本 | 客户懒得换 | SAP、Oracle | 很长（替换太麻烦） |
| 牌照/特许经营权 | 政府只准你干 | 银行、证券、电信 | 取决于政策 |
| 资源优势 | 独占稀缺资源 | BHP、紫金矿业 | 有限（资源会枯竭） |
| 地理优势 | 位置决定了 | 港口、收费公路 | 无限（位置不会变） |

### 2. 判断壁垒强弱的三条标准

**第一条：定价权**
- 公司能涨价吗？
- 涨价后客户会流失吗？
- → 敢涨价且客户不跑的，壁垒很厚

**第二条：超额利润**
- ROE是否持续>15%？（去掉杠杆后也高）
- 毛利率是否稳定或提升？
- → 能长期赚比别人多的钱，说明有壁垒

**第三条：时间检验**
- 这个壁垒5年后还在不在？
- 技术壁垒会被颠覆吗？（诺基亚的教训）
- → 品牌和转换成本通常比技术壁垒更持久

### 3. 壁垒趋势更重要

```
壁垒在加厚 ←──────────→ 壁垒在被侵蚀

        好现象                     坏现象
  - 毛利率持续提升             - 毛利率持续下降
  - 市占率持续上升             - 市占率被蚕食
  - 客户越来越依赖             - 客户开始用替代品
  - 竞争对手退出               - 新玩家不断涌入
```

## 三、驱动力拆解：靠量还是靠价

### 1. 所有收入都可以拆解

```
收入 = 量 × 价

利润 = 量 ×（价 - 单位成本）
```

### 2. 不同行业的"量"和"价"

| 行业 | 量的指标 | 价的指标 | 关键变量 |
|------|---------|---------|---------|
| 证券 | 日均成交额、两融余额 | 佣金率、净息差 | 市场活跃度 |
| 光伏 | 装机量（GW） | 组件价格（元/W） | 硅料价格+补贴政策 |
| 金矿 | 黄金产量（oz） | 金价（AUD/oz） | 美元实际利率 |
| 银行 | 贷款规模 | 净息差 | 利率周期+信贷需求 |
| 超市 | 同店销售增长 | 毛利率 | 居民消费+竞争 |
| SaaS | 客户数×ARPU | 续费率 | 获客成本+产品力 |

### 3. 定量判断：收入变动中多少来自量、多少来自价

```
收入增速 ≈ 量增速 + 价增速（粗略）
收入增速 =（1+量增速）×（1+价增速）-1（精确）

案例（东方财富2025）：
营收+38.46%
量（股基成交额）：+58.66%
价（佣金率）：估算下降约13%
→ 量贡献了主要增长，价在下降（行业佣金率长期下行趋势）
```

**四种组合：**

| 量 | 价 | 判断 |
|---|---|---|
| ↑ | ↑ | ⭐ 最理想——量价齐升（戴维斯双击） |
| ↑ | ↓ | 常见——以价换量（需确认量能不能持续弥补价跌） |
| ↓ | ↑ | 涨价撑收入——不可持续（需求萎缩后价也会跌） |
| ↓ | ↓ | 最差——量价齐跌（避开） |

### 4. 成本结构拆解

```
利润 = 收入 - 成本

成本结构决定了利润弹性：
```

**固定成本高 vs 变动成本高**

| | 固定成本高（重资产） | 变动成本高（轻资产） |
|--|-------------------|-------------------|
| 典型行业 | 钢铁、化工、航空、矿业 | 软件、服务、贸易 |
| 收入下降10% | 利润下降>20%（杠杆效应） | 利润下降~10% |
| 收入上升10% | 利润上升>20%（利润弹性大） | 利润上升~10% |
| 盈亏平衡点 | 高，开工率必须达到~70% | 低，卖一单赚一单 |

> 重资产行业在周期底部的亏损可以极其惨烈（固定成本还在，收入没了）。但在复苏期利润弹性也最大。这就是周期性行业的双刃剑。

### 5. 单位经济学（Unit Economics）

适用于分析一家公司的单笔生意赚不赚钱：

```
证券行业：
每笔交易收入 = 佣金费率 × 交易金额
每笔交易成本 = 交易所规费 + 系统成本分摊 + 获客成本摊销

天天基金（东方财富）：
每客户收入 = 客户AUM × 管理费率
每客户成本 = 获客成本（CPA）÷ 客户生命周期 + 运维成本
```

> 如果单位经济为正（单笔生意赚钱），公司规模越大越赚钱。如果为负，规模越大亏越多（很多烧钱的公司就是这个原因）。

## 四、L9输出：驱动力仪表盘

每研究一个行业，完成这张表：

```
┌─────────────────────────────────────────────────┐
│              驱动力与竞争速查卡                     │
├─────────────────────────────────────────────────┤
│ 收入公式：________ × ________                     │
│ 关键变量1：________  最新值：____  历史区间：____  │
│ 关键变量2：________  最新值：____  历史区间：____  │
├─────────────────────────────────────────────────┤
│ 市场结构：______竞争  CR3：____%  趋势：____      │
│ 格局象限：□赢家通吃 □百花齐放 □寡头稳定 □价格战    │
├─────────────────────────────────────────────────┤
│ 核心壁垒（top 2）：________ + ________            │
│ 壁垒趋势：□在加厚 □稳定 □在被侵蚀               │
├─────────────────────────────────────────────────┤
│ 成本特征：□重资产（固定成本高） □轻资产（变动成本高）│
│ 利润弹性：营收±10% → 利润±____%（估算）          │
└─────────────────────────────────────────────────┘
```

---

> 💡 L9的核心输出是**行业的"收入公式"**和**竞争格局的定位**。学完L9，你应该能用一句话说清楚："这个行业赚钱靠XX，龙头靠XX壁垒守住利润。"带着这个结论进入L10判断买卖时机。
