# L4 - 杜邦分析概念题（答案）

## 题1

**答案：B. 24%**

ROE = 8% × 1.5 × 2.0 = 24%

**易错点：** 计算时先乘利润率和周转率（8%×1.5=12%），再乘权益乘数（12%×2=24%）。不要搞混次序。

## 题2

**答案：B. 甲最优质——利润率高、杠杆低，护城河深**

分析：
- 甲：ROE = 30%×0.5×1.3 = 19.5% → 靠高利润率驱动，杠杆最低，最可持续
- 乙：ROE = 5%×2.0×1.8 = 18% → 高周转型，运营压力大
- 丙：ROE = 15%×0.4×3.0 = 18% → 杠杆偏高，负债率~67%，经济下行风险大

## 题3

**答案：B. 资产周转率大幅下降（新增资产未产生预期营收）**

分析：
- 净利润率：16%→14%，下降2pct，小幅影响
- **周转率：1.2→0.7，下降42%，是核心拖累因素**
- 权益乘数：1.04→1.22，上升（杠杆增加了），反而推高ROE，但因周转率下降太猛没补回来

结论：公司可能投了很多钱建厂/收购，但营收增长没跟上资产增速。

## 题4

**答案：B. 净利润率 2%、周转率 2.5、权益乘数 3.5**

分析：
- B的ROE = 2%×2.5×3.5 = 17.5%
- 净利润率极薄（2%），杠杆极高（3.5→负债率~71%）
- 一旦营收下滑，薄利润+高杠杆会让ROE迅速归零甚至亏损
- A是典型高利润型（优质）；C是均衡型；D是合理高周转型

## 题5

**答案：D. 营收+20%，净利润率稳定在 4.5%，资产周转率从 1.6 升到 1.8**

分析：
- A：净利润率下降+杠杆上升 → 以价换量+借钱扩张，危险
- B：周转率下降+净利率微升 → 新店效率低于老店，改善不明显
- C：杠杆从1.6升到3.5（负债率~71%）→ 扩张全靠借钱，风险大
- D：营收增+周转率提升 → 新店比老店效率更高，净利率稳定 → 健康的规模效应
