核心结论：

② ROE从2021年33%回落到2025年12.44%，是周期性的利润回归而非竞争力丧失——公司销量一直在增长（近5年营收增长近一倍）；

六、行业分类与特征（L5框架）

6.1 行业类型判断

行业类型：纯周期股（基础化工—聚氨酯/石化）
判断依据：万华化学主营MDI/TDI（聚氨酯原料）和石化产品，均为典型的大宗化工品。MDI行业高度成熟，需求增速约GDP+1~2%（约3-4%/年），长期不具高成长性，但利润波动极大——2021年MDI价差高峰时万华年净利246亿，到2025年价差低迷时净利125亿，波动幅度近50%。纯周期行业属性：供需错配驱动的价格周期才是利润波动的核心变量，而非量的增长。

周期特征确认：万华过去5年ROE在12%-33%之间剧烈波动，完全符合周期股特征。本次周期触底（2024-2025）后，2026Q1已出现明确修复信号。

6.2 行业集中度与壁垒

维度判断
行业集中度极度集中——全球MDI CR3（万华/巴斯夫/科思创）>80%，万华中国市占率>50%。全球有效产能约1,000万吨/年，万华约310万吨，且拥有全球最低成本。
技术壁垒极高——MDI生产涉及光气化反应，工艺极其复杂，全球仅4家（万华+巴斯夫+科思创+亨斯迈）拥有大规模量产技术。新进入者几乎不可能10年内建成有竞争力的装置。
规模壁垒极高——单套MDI装置投资超100亿元，且必须配套氯碱、苯胺等上下游，整个一体化基地投资超500亿元。规模门槛足以阻止绝大多数新进入者。
成本壁垒极强——万华已做到全球成本最低（烟台/宁波基地的原料自供+能源配套一体化+循环经济），竞争对手（巴斯夫/科思创）在高成本欧洲的装置正在缩产。

6.3 生命周期阶段

阶段：成熟期（MDI）+ 成长期（新材料）
聚氨酯（MDI）是成熟行业——全球需求平稳增长3-4%/年，行业格局已定，CR3>80%。万华在这一阶段要做的是"以量换利"——通过扩产和降本挤出高成本竞争对手。精细化学品及新材料处于成长期（+17.39%），是万华的第二增长曲线，正在从~15%毛利率向更高附加值方向演进。

投资策略重点：MDI看的是产能出清节奏和价差修复（边际产能退出的速度决定利润率修复的幅度）；新材料看的是技术突破和客户导入进度（电池材料/ADI/尼龙12等）。

6.4 当前周期位置与判断

当前周期位置：底部已过 → 复苏初期（2026Q1已经确认）
判断依据：① 2025年全年化工品价格持续下行（超90%品类均价同比下跌），2025利润125亿已较2021年峰值246亿腰斩有余；② 2025Q4营收+71%、净利+33.7亿（同比+73.7%）——新装置放量+MDI价格触底回升；③ 2026Q1净利37.2亿（同比+20.6%、环比+10.3%）进一步确认盈利修复趋势；④ 从行业整体看，聚氨酯价差在2025年底开始走阔，石脑油-PX-PTA链条也有改善。

判断对评分的影响：周期位置维度（15分权重）可得12分（底部已过复苏初期）——高于底部纯左侧时但低于中周期的确定性，是目前评分体系中最核心的加分项。

6.5 估值方法匹配

行业类型首选估值辅助指标当前是否匹配
周期型（基础化工）PB（周期性底部）PE（顶部参考）、产能利用率、产品价差✅ 匹配——PB 2.5x处于27%分位，估值偏低
估值方法总结：对于万华化学这类纯周期股，使用PB估值是唯一正确的框架。当前PB 2.5x处于5年27%分位，属于底部区域。PE 21x看似"偏贵"（5年88%分位），但这是利润已跌至底部的正常现象——2025年利润较峰值下降63%才得到这个"高PE"。当利润修复到市场中性的2026年18,307亿预测时，对应PE仅15x。周期股投资是"PB在底部买入、PE在低位时等待、PE升高时卖出（利润已反映）"的逆向逻辑。当前阶段，PB底部+拐点已现=左侧布局窗口。

七、综合实战（L6框架）

7.3 三个陷阱排查

☐ 价值陷阱：❌ 未命中——低PE（看似低）+低增长（周期底部）+高负债（62.8%）的组合恰恰是周期股底部的典型特征。价值陷阱的精髓是"低PE+利润不可持续"，万华的利润周期属性决定了需要用PB而非PE来判断。
☐ 周期陷阱：⚠ 注意但不命中——周期陷阱发生在"低PE+周期顶部"，即利润暴利期PE极低时买入。当前处于周期底部（利润下跌63%后企稳回升），底部买入是正确操作而非陷阱。但需注意复苏可能慢于预期。
☐ 烧钱陷阱：❌ 未命中——高PE + 高增速但CFO为负才是烧钱陷阱。万华CFO 331亿极其强劲，FCF 80亿为正，完全不涉及烧钱问题。

7.6 关键变量敏感性

变量1: MDI价差（核心盈利变量）
乐观: 海外高成本产能加速退出+需求复苏 → 价差扩大20% → EPS增加~1.5元
中性: 价差温和修复至历史均值 (已体现在券商2026E中) → EPS 5.87元
悲观: 全球经济衰退 + 新增产能冲击 → 价差维持2025年水平 → EPS 3.5-4.0元

中性: 维持~15%毛利率，营收稳定增长 → 对EPS贡献边际改善
悲观: 竞争加剧/导入不及预期 → 毛利率压缩至10% → 拖累整体盈利

变量3: 石化板块盈利
乐观: 乙烷裂解成本优势持续兑现 + 石化品价格修复 → 毛利率提升至8-10%
中性: 维持~5%毛利率，靠量增长 → 不贡献利润但也不亏

7.7 前景展望

正向驱动：
1. MDI价差修复 — 全球高成本产能收缩（欧洲能源成本高企）+ 万华持续扩产抢占份额，价差有望持续修复
2. 乙烷裂解成本优势兑现 — 蓬莱POCHP装置开车成功，相较于石脑油裂解持续降低成本，石化板块盈利改善
3. 新材料放量 — 精细化学品+17.39%增长，电池材料（磷酸铁锂/钠电）和多款ADI/尼龙12产品放量驱动第二曲线

负向拖累：
1. 全球化工需求恢复不确定 — 房地产、汽车等下游修复节奏仍有变数，若复苏低于预期则P上涨乏力
2. 折旧和资本开支压力 — 每年~250亿资本开支+折旧134亿，若利润率修复不及预期则FCF将收缩
3. 贸易摩擦 — 美国关税升级和地缘政治风险可能影响海外业务（海外营收占比47%）
