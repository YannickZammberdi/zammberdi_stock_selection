## 1. 怎么判断指标高和低

> ★★★ [L1]

没有固定答案，但有三种判断方法。

### 跟同行比（最常用）

同一个行业中，毛利率低于同行均值 → 产品竞争力弱。
同行业中，三费占比高于同行 → 运营效率低。

```
以白酒行业为例（2023年数据）：
  茅台的毛利率：91%
  五粮液的毛利率：76%
  某个小酒企：40-50%

如果一家白酒企业毛利率只有40%，说明它没有品牌溢价，只能卖低价酒。
```

### 跟自己历史比（看趋势）

即使绝对值"看起来可以"，逐年下滑就是危险信号。

```
某消费电子公司毛利率变化：
  2021年：28%
  2022年：25%
  2023年：20%
  趋势：连续下滑 → 产品竞争力在减弱，或被竞争对手挤压
```

### 跨行业认知基准（建立直觉）

#### 毛利率参考

| 范围 | 含义 | 典型行业 |
|------|------|----------|
| 60%+ | 极高毛利，强定价权 | 奢侈品、高端白酒、软件 |
| 30-60% | 中等偏高，有一定壁垒 | 医药、家电、汽车零部件 |
| 10-30% | 中等偏低，竞争激烈 | 零售、制造、物流 |
| <10% | 极低，靠规模/周转 | 超市、大宗商品贸易 |

#### 三费占比参考（销售+管理+研发费用 ÷ 营收）

| 范围 | 含义 | 典型 |
|------|------|------|
| <10% | 费用控制好，或品牌强不需要推广 | 茅台（约9%） |
| 10-20% | 正常水平 | 大多数制造业公司 |
| 20-30% | 偏高，需关注是否在烧钱 | 科技公司、药企（研发投入大） |
| >30% | 非常高，需要警惕 | 部分亏损的互联网公司 |

**重要提醒：** 不要死记硬背数字。同样的毛利率，在白酒行业属于很差，在零售行业却是顶级。先确定行业，再比同行，再看趋势。

## 2. 归母公司净利润是什么

> ★★★ [L1]

**全称：归属于母公司股东的净利润**

> 简单说：上市公司合并报表里，扣掉"子公司里其他股东该分走的部分"之后，真正属于**上市公司股东**的利润。

### 为什么会有这个概念？

上市公司通常控股多个子公司，但未必持有100%股权。

```
假设：A上市公司控股子公司B（持股70%）
B公司今年赚了1亿
→ B的1亿利润全部合并在A的报表里
→ 但其中3000万属于B的30%少数股东（不是你）
→ A需要扣除这3000万，剩下7000万才是"归母公司净利润"
```

**关键：看上市公司股票时，"归母公司净利润"才是衡量你这份股票值多少钱的依据。** 合并净利润里包含了不属于你的部分。

## 3. EBITDA和归母公司净利润一样吗

> ★★ [L1→L2]

**完全不一样。** 它们回答完全不同的问题：

| | EBITDA | 归母公司净利润 |
|--|--------|--------------|
| 回答 | 经营上赚了多少？ | 真正属于股东的钱有多少？ |
| 扣除了什么？ | 利息/税/折旧/摊销 | 少数股东权益 |
| 没扣除什么？ | 折旧/利息/税 | 利息/税/折旧 |
| 用途 | 看经营能力 | 看股东到手价值 |

```
EBITDA高 ≠ 归母净利润高

例如：
  某电信公司 EBITDA=100亿（经营不错）
  利息20亿 + 折旧40亿 + 税10亿
  → 净利润=30亿
  → 其中10亿是少数股东的
  → 归母公司净利润=20亿（才是你的）
```

## 4. 毛利率和三费的高低怎么看

> ★★★ [L1]

**毛利率高低**取决于行业属性+公司竞争力：
- 高毛利率（60%+）：通常意味着产品有品牌溢价、技术壁垒或垄断地位
- 低毛利率（<20%）：通常是竞争充分的商品化行业，或零售/贸易

**三费高低**取决于商业模式：
- 茅台三费低（~9%）：品牌力强，不需要打广告
- 恒瑞医药三费高（~40%+）：大量投入研发和销售推广
- 关注趋势：三费占比逐年上升，说明获客成本越来越高，不是好兆头

**毛利率 × 周转率 = 最终利润**

> 茅台的逻辑：高毛利率 × 低周转
> 永辉的逻辑：低毛利率 × 高周转
> 两种模式都可以赚钱，关键是管理效率

## 5. 扣非净利润是什么为什么比净利润重要

> ★★ [L1→L2]

**扣非净利润 = 净利润 - 非经常性损益**

非经常性损益包括：
- 卖资产（卖厂房、卖子公司、卖地）
- 政府补贴
- 投资收益（炒股、理财）
- 债务重组收益
- 资产减值转回

**为什么重要：**

```
公司A：
  净利润：20亿（看起来不错）
  其中卖地赚了10亿，政府补贴5亿
  → 扣非净利润只有5亿
  → 明年没有地卖，利润会暴跌

公司B：
  净利润：8亿（看起来不如今年的A）
  扣非净利润：7.5亿
  → 利润几乎全来自主业
  → 更可持续
```

**判断原则：**
- 扣非净利润持续增长 → 主业在变好
- 净利润增长但扣非停滞 → 靠卖资产/补贴撑业绩
- 扣非净利润与经营现金流不匹配 → 警惕（纸面富贵）

## 6. 需要仔细拆解营业成本的构成吗

> ★★ [L1]

**对大多数投资者来说，不需要。**

营业成本分三类：原材料、人工、制造费用（折旧、水电等）。拆到这么细一般是分析师或者管理层该做的事，投资者只需要关注一件事——**毛利率的趋势和原因**。

### 什么时候需要往下挖？

**情况一：毛利率突然大幅下降**

```
比如某家电公司毛利率从30%跌到22%
→ 需要问：是因为原材料涨价？还是产品在降价？
→ 如果是原材料涨价（外部因素），能否转嫁给下游？
→ 如果是产品降价（竞争加剧），这是结构性问题
```

**情况二：公司之间的毛利率差异需要解释**

```
两家钢铁公司：
  A公司毛利率25%，B公司毛利率15%
→ 差异可能来自：
   - A有上游矿山，原料成本更低
   - A的产品更高端（汽车板 vs 螺纹钢）
   - A的管理效率更高
```

### 一句话原则

> 从毛利率变化去倒推成本端的压力，比直接拆原材料要高效得多。**"毛利率"已经够用了，除非你怀疑毛利率突然变化的原因。**

## 7. EBITDA是什么和净利润有什么区别

> ★★★ [L1]

**EBITDA = 息税折旧摊销前利润**

```
EBITDA = 净利润 + 所得税 + 利息 + 折旧 + 摊销
```

### 为什么需要这个指标？

净利润包含了太多"非经营因素"（利息是融资决策，折旧是会计估计，所得税是政策因素），EBITDA试图还原公司纯粹靠**经营业务**赚了多少钱。

### 和净利润的区别

```
某制造公司：
  净利润：5亿
  + 折旧和摊销：3亿（厂房设备每年贬值）
  + 利息费用：1亿（银行贷款利息）
  + 所得税：1亿
  EBITDA：10亿
```

| | 净利润 | EBITDA |
|--|--------|--------|
| 包含折旧 | 是（扣掉了） | 否 |
| 包含利息 | 是 | 否 |
| 包含税 | 是 | 否 |
| 反映什么 | 股东到手的钱 | 经营产生的钱 |

### 什么时候看EBITDA？

- **重资产行业**（钢铁、水泥、电信）：折旧巨大，净利润被严重压低，用EBITDA更能反映真实经营水平
- **高负债公司**：利息负担重，EBITDA能看出"不考虑利息情况下赚了多少"
- **收购估值**：常用EV/EBITDA来估值

### 什么时候EBITDA没用？

- **轻资产公司**（软件、白酒）：折旧几乎没有，EBITDA和净利润差不多
- **初创亏损公司**：EBITDA为负还看它干嘛

## 8. 营业利润和净利润有什么区别

> ★★ [L1→L2]

**营业利润**是公司靠主营业务赚的钱，**净利润**是营业利润加上"其他乱七八糟"之后的最终数字。

```
营业利润 = 营收 - 成本 - 三费 - 折旧 - 减值
  + 其他收益（政府补贴等）
  + 投资收益
  + 公允价值变动（持有的股票涨跌）
  - 资产减值
  = 利润总额
  - 所得税
  = 净利润
```

**关键判断：**

| 情况 | 含义 |
|------|------|
| 营业利润 = 净利润的90%+ | ✅ 利润干净，来自主业 |
| 净利润靠投资收益撑着 | ⚠️ 主业可能不行，靠理财炒股撑利润 |
| 净利润靠公允价值变动 | ❌ 持有其他公司股票涨跌造成的，不可持续 |

**一个常见陷阱：** 公司持有某上市公司股票，股价大涨→公允价值变动收益暴增→净利润很好看→但这是纸面富贵，股价跌了会全部吐回去。

## 9. 为什么有些公司利润很高却不分红

> ★★ [L1]

常见原因：

1. **处于高速成长期** — 赚钱要全部再投入扩张（如早期的亚马逊、宁德时代）
   - 股东赚的是股价上涨，不是分红
2. **账面利润不是真金白银** — 利润停留在应收账款或存货上，没有现金可分
   - 检查经营现金流就知道了
3. **有大量资本开支需求** — 每年赚的钱必须拿去更新设备、建厂房
   - 自由现金流少，自然分不出
4. **偿还债务优先** — 利润优先还贷，不分红

**判断：** 如果一家成熟公司（非高成长期）利润高但不分红，检查它的自由现金流。

## 10. 利润表常见陷阱

> ★★★ [L1]

### 10.1 研发费用资本化

> 研发投入有两种记账方式：
> - **费用化**：当年从利润表里扣掉（保守）
> - **资本化**：记成无形资产，分期摊销（激进）

两家研发投入相同的公司，选择"资本化"的那家当期利润会更高——但这只是会计手法的差异，不是真实盈利差异。

**怎么防：** 同行业对比时，注意它们是否采用相同的研发处理方式。

### 10.2 营收确认操纵

> 公司可以提前确认收入来美化当期业绩。

**常见手法：**
- 货物还没发出就确认收入
- 把产品卖给经销商但允许退货（"压货"）
- 长期合同按工程进度确认收入（可以人为调节进度）

**怎么防：** 结合经营现金流判断——营收大增但经营现金流没同步增长，警惕。

### 10.3 一次性费用反复出现

> 有些公司每年都出现"一次性重组费用"、"资产减值"、"诉讼损失"。

如果"一次性"费用每年都有，那就不是一次性的——说明公司经营本身就有问题。

**怎么防：** 看过去3-5年，是否每年都有大额"非经常性损失"。

## 11. 资产负债率多少算安全

> ★★★ [L1]

取决于行业：

| 行业 | 安全范围 | 说明 |
|------|----------|------|
| 白酒/消费 | <30% | 现金多债务少才是好 |
| 制造业 | 40-60% | 需要借钱建厂，正常 |
| 房地产 | 70-90% | 高杠杆行业，重点关注现金流能否覆盖利息 |
| 银行 | 90%+ | 银行本质就是吸收存款放贷款，天生高负债 |

**更重要的不是比例，而是：**
- 借的钱拿去干什么了？（投资赚钱的业务 vs 填补亏损）
- 经营现金流能覆盖利息吗？（利息保障倍数）
- 债务结构如何？（短期多还是长期多？）

## 12. 有息负债和无息负债有什么区别

> ★★★ [L1→L2]

**这是判断一家公司真实负债压力的关键。**

| | 有息负债 | 无息负债 |
|--|----------|----------|
| 包括 | 短期借款、长期借款、应付债券 | 应付账款、合同负债（预收）、应付职工薪酬、应交税费 |
| 代价 | 要还本付息 | 不需要利息 |
| 议价含义 | 公司缺钱才去借 | 占用了上下游的资金，说明有话语权 |

**实战案例：**

```
公司A：总负债100亿
  有息负债：80亿（大量银行贷款）
  无息负债：20亿

公司B：总负债100亿
  有息负债：10亿（借款很少）
  无息负债：90亿（大量欠供应商+预收客户款）

→ 同样的资产负债率，B的财务压力远小于A
```

**快速判断：** 资产负债率不是重点，**有息负债率**才是。

## 13. 应收账款高一定不好吗

> ★★★ [L1]

不绝对，分情况：

**不好：**
- 应收账款增速 > 营收增速 → 说明靠放宽账期换增长
- 应收账款/营收 > 30-50%（视行业而定）
- 账龄长（1年以上应收占比高） → 收回可能性低
- 客户集中度高（几个大客户欠了大部分钱）

**可以接受：**
- 行业惯例（如工程机械、医疗器械常用分期付款）
- 客户都是信誉良好的大型企业/政府
- 应收账款周转天数在行业正常范围内

**最好：** 应收账款很低且账期短 → 说明对下游议价能力强
- 茅台、海天味业都是先款后货

## 14. 折旧和摊销为什么也能影响资产负债表

> ★★★ [L1]

> 折旧/摊销是连接**三张表**的关键环节。

**一张图看懂：**

```
利润表：折旧费用 → 扣减净利润
      ↓
资产负债表：固定资产账面价值减少（累计折旧增加）
      ↓
现金流量表：折旧不影响现金流（在CFO里加回）
```

**所以：**
- 折旧在**利润表**上扣了利润
- 但在**现金流表**上加回来了（因为没真花现金）
- 同时**资产负债表**上的固定资产逐年减少

**关键洞察：**

```
净利润被折旧压低 → PE变高（看起来贵）
但CFO不受影响 → PCF不变（实际估值没变）

所以重资产公司看PE会"失真"，要结合PCF或EBITDA看。
```

> PCF（市现率）= 总市值 ÷ 经营现金流。比PE更难造假，因为经营现金流比净利润更难操纵。L3会系统讲所有估值指标。

## 15. 资产负债表常见陷阱

> ★★ [L1]

### 15.1 商誉减值

> 公司收购另一家公司时，多付的钱记作"商誉"。

如果收购来的公司业绩不达标，商誉就要**减值**——直接从利润表里扣掉，可能导致当年利润暴跌甚至亏损。

```
某公司收购了一个品牌，溢价10亿记作商誉
收购后发现品牌价值下降
→ 商誉减值5亿，当年净利润直接少5亿
→ 股价可能一天跌20%
```

**怎么防：** 看资产负债表上的"商誉"占净资产的比重。如果商誉/净资产>30%，就要当心。

### 15.2 存货跌价风险

> 存货可能因为过时/变质/价格下跌而价值缩水。

| 行业 | 风险 |
|------|------|
| 电子/服装 | 产品更新快，过季就大幅贬值 |
| 生鲜食品 | 保质期短，卖不掉就报废 |
| 房地产 | 房价下跌时存货价值缩水 |
| 白酒 | 越陈越值钱，风险低 |

**怎么防：** 看"存货跌价准备"计提是否充分，以及与同行的对比。

### 15.3 大股东质押

> 大股东把持有的股票质押给银行换取贷款。

如果股价下跌到警戒线，银行要求补充质押或被强制平仓 → 股价会进一步暴跌。

**怎么防：** 查公告中"股东质押"信息，质押比例>50%需要警惕。

## 16. 资产负债表各科目安全范围速查

> ★★ [L1→L2]

下表供快速参考（具体要结合行业+同行对比）：

| 科目 | 怎么看 | 安全范围 | 危险信号 |
|------|--------|----------|----------|
| 货币资金 | 能否覆盖短期借款 | 货币资金 > 短期借款 | 不够还短期借款，流动性风险 |
| 应收账款 | 占营收比例 + 账龄 | <30%营收，账龄<1年 | 增速>营收增速，或占比>50% |
| 存货 | 周转天数 + 跌价风险 | 周转稳定，无大量积压 | 周转天数逐年上升，或行业下行 |
| 固定资产 | 占总资产比例 | 看行业（重资产<60%，轻资产<30%） | 产能利用率低时固定资产占比过高 |
| 商誉 | 占净资产比例 | <30% | >50%就要高度警惕减值风险 |
| 短期借款 | 与货币资金对比 | 货币资金能覆盖 | 短期借款>货币资金 |
| 有息负债率 | 有息负债 ÷ 总资产 | 行业均值以内 | 远超同行 |
| 合同负债（预收） | 趋势 | 逐年增长最好 | 逐渐下降→产品需求减弱 |
| 应付账款 | 趋势 | 稳定或略增（占上游资金） | 暴增→可能付不起供应商钱 |
| 未分配利润 | 正还是负 | >0 | <0说明历年累计亏损 |
| 净资产 | 正还是负 | >0 | <0资不抵债，濒临退市 |

**核心逻辑：**
- 资产端：看有没有"虚"的资产（应收高不高、商誉大不大、存货会不会跌价）
- 负债端：看有没有"急"的债（短期借款多不多、利息能不能覆盖）
- 权益端：看股东的钱是多了还是少了（未分配利润、净资产趋势）

## 17. 经营现金流和净利润到底有什么区别

> ★★★ [L1]

核心区别：

| | 净利润 | 经营现金流 |
|--|--------|-----------|
| 基础 | 权责发生制 | 收付实现制 |
| 含义 | 赚了多少（会计上） | 收到了多少现金 |
| 可操纵性 | 较高（通过会计估计） | 较低 |

**正常情况：经营现金流 ≈ 净利润（略高因为折旧是非现金支出）**

**危险情况：净利润 > 经营现金流，且差距在扩大**
- 可能原因：应收账款激增（货卖了钱没到）、存货积压
- 这是"纸面富贵"的信号

**极端健康：经营现金流 > 净利润**
- 说明公司不仅有真金白银，还有预收账款（先收钱后交货）
- 茅台就是典型

### 17.1 折旧完了不是要买新设备吗

> ★★ [L1] 那CFO>净利润还值得高兴吗？

**这是一个非常好的问题。** 你抓住了关键矛盾：

```
折旧是非现金支出 → CFO > 净利润（虚高）
但设备确实在老化 → 将来要花钱换
```

**解法：看自由现金流（FCF = CFO - 资本开支）**

```
某制造公司：
  净利润：10亿
  折旧：5亿
  CFO：15亿（净利润+折旧）
  资本开支：12亿（买新设备、维护旧设备）
  
  FCF = 15 - 12 = 3亿  ← 这才是真正能自由支配的钱
```

| 情况 | 含义 |
|------|------|
| FCF > 0 | 扣除维持经营的投资后，还有剩余 |
| FCF = 0 | 挣的钱刚好够换设备，没有多余 |
| FCF < 0 | 挣的钱不够投资，需要借钱或融资 |

**所以结论：**
- CFO > 净利润，如果**资本开支也很大** → 钱只是"过路财神"，最终要花出去换设备
- CFO > 净利润，且**资本开支远小于折旧** → 这才是真正的好（如茅台，折旧很小，资本开支也很小）
- **最终要看FCF，不是CFO**

### 17.2 预收款多永远是好事吗

> ★★ [L1]

**多数情况下是好事，但不绝对。**

**好事的情况：**
- 先收钱后发货 → 说明产品抢手，有定价权
- 占用客户资金 → 变相零成本融资
- 茅台、海天、微软都是典型

**需要警惕的情况：**

| 情况 | 风险 |
|------|------|
| 预收款增速远高于营收增速 | 可能产品供应紧张，客户被迫提前付款；或公司有意推迟确认收入 |
| 房地产公司 | 卖期房收到的预收款，如果项目烂尾，钱要退回去，房子还交不了 |
| 教育培训 | 预收学费（合同负债巨大），但服务周期长，如果爆雷退费压力极大 |
| 预收款占比过高 | 公司极度依赖客户预付款来维持现金流，一旦客户信心动摇，现金流会瞬间断裂 |
| 刚上市/新公司预收款暴涨 | 可能是通过放宽退款政策吸引客户，实际交付能力存疑 |

**判断方法：** 预收款 vs 历史趋势 vs 行业惯例。健康的预收款是产品力强的结果，而不是公司刻意追求的目标。

## 18. 不同公司的现金流模式有哪些

> ★★★ [L1→L5]

不同阶段的公司，三张表的现金流特征完全不同。以下是7种常见模式：

### 模式A：现金奶牛（消费龙头）

```
典型：茅台、海天味业、可口可乐
CFO：强正数，且>净利润
CFI：小负数（少量资本开支）
CFF：大负数（大量分红+回购）
```

特点：每年产生大量现金，但不需要大量再投资。成熟期，是价值投资者最爱的类型。

### 模式B：高成长烧钱期（早期成长股）

```
典型：早期的亚马逊、拼多多、未盈利生物科技
CFO：负数或微正
CFI：大负数（大量投资扩张）
CFF：大正数（不断融资）
```

特点：靠融资续命，赌的是将来能盈利。如果融资环境恶化，可能断粮。

### 模式C：资本开支吞噬现金流（重资产制造）

```
典型：宁德时代、台积电、京东方
CFO：大正数
CFI：大负数（建厂买设备，>CFO）
CFF：正数（需要借款或增发）
```

特点：赚的钱全部拿去投资，自由现金流很少甚至为负。**赚到的不是能花的钱。**

### 模式D：借新还旧（高杠杆企业）

```
典型：房地产公司、部分基建企业
CFO：不稳，周期性强
CFI：大负数（持续买地）
CFF：大正数（不断借新债）
```

特点：现金流健康的核心指标是"经营现金流能否覆盖利息"。

### 模式E：稳定收租（公用事业）

```
典型：长江电力、燃气、高速公路
CFO：强正数，极其稳定
CFI：大负数（偶尔大修或扩建）
CFF：正数（借款）+ 负数（分红）
```

特点：可预测性极高，适合收股息。

### 模式F：周期过山车（资源周期股）

```
典型：宝钢、紫金矿业、中远海控
好年景：CFO大正数，CFI大负数（扩产），CFF还债+分红
差年景：CFO极低或负，CFI停止投资，CFF借新债
```

特点：现金流随经济周期剧烈摆动。

### 模式G：整合收购（并购驱动型）

```
典型：爱尔眼科、药明康德（扩张期）
CFO：正数
CFI：大负数（不断收购公司）
CFF：正数（靠融资收购）
```

特点：增长靠收购而非内生。风险是商誉越积越多。

### 快速判断表

| 类型 | CFO | CFI | CFF | 自由现金流 | 典型阶段 |
|------|-----|-----|-----|-----------|---------|
| 现金奶牛 | +++ | - | -- | 丰厚 | 成熟期 |
| 高成长烧钱 | -/0 | -- | ++ | 负 | 成长期 |
| 重资产制造 | ++ | --- | + | 少或负 | 扩张期 |
| 高杠杆企业 | +/- | -- | ++ | 负 | 永续 |
| 公用事业 | +++ | - | +/- | 稳定 | 成熟期 |
| 周期股 | 波动大 | 波动大 | 波动大 | 波动大 | 随周期 |
| 并购驱动 | + | --- | ++ | 负 | 整合期 |

### 判断步骤

```
1. CFO是正还是负？
   → 负：公司在烧钱，需要融资续命
   → 正：进入下一步

2. 自由现金流（CFO-CFI）是正还是负？
   → 正且丰厚：现金奶牛或公用事业
   → 负：钱投到了哪？
     → 固定资产 → 重资产制造
     → 收购 → 并购驱动
     → 存货/买地 → 高杠杆

3. CFF是正还是负？
   → 正：公司在借钱或增发
   → 负：公司在还债或分红
```

## 19. 其他常见陷阱

> ★★ [L1]

### 19.1 关联交易

> 公司和关联方（大股东控制的另一家公司）之间的交易。

**猫腻：**
- 高价卖给关联方 → 虚增营收和利润
- 低价从关联方买资产 → 利益输送
- 把钱借给关联方 → 资金被占用

**怎么防：** 在财报里找"关联交易"章节。如果关联交易金额大、定价不合理，需要警惕。

### 19.2 周期股PE陷阱

> 周期股利润最高的时候PE最低（看起来"便宜"），恰恰是股价顶部。
> 周期股亏损的时候PE为负（看起来"贵"），往往是买入时机。

**核心：周期股不能用PE估值，要用PB。**

### 19.3 高送转/拆股

> 1股拆成10股，股价从100变成10，总价值不变。

没有任何实质意义，但A股市场有时会炒作"高送转"题材——纯粹的投机。

**怎么防：** 不看这些噱头，只关注公司基本面。

### 19.4 财务对比口径不一致

> 公司变更了会计政策或合并范围，导致今年和去年的数字不可比。

**例如：**
- 收购了一家公司 → 今年合并报表多了这家公司的营收（不是内生增长）
- 改变了折旧年限 → 折旧变少、利润变高

**怎么防：** 看财报"会计政策变更"附注，比较时用"同口径"数据。
