## 1. 高ROE的公司都是现金奶牛吗

**不一定。** ROE高有三个不同的来源，只有一种是现金奶牛。

```
ROE = 净利润率 × 资产周转率 × 权益乘数
```

### 真正的高ROE现金奶牛

靠**极高的净利润率**驱动，杠杆很低，自由现金流丰厚。

```
茅台 ROE=30%：
  净利润率 49%  ← 核心驱动力
  周转率 0.5次
  权益乘数 1.23（几乎无杠杆）
  FCF/净利润 > 1（现金质量好，大量分红）
```

### 高ROE但不是现金奶牛的三种反例

| 类型 | 高ROE的原因 | 为什么不是现金奶牛 |
|------|-----------|----------------|
| 银行 | 杠杆极高（权益乘数>12） | 赚的钱必须保留为资本金应对监管，不能随便分，且FCF低 |
| 房地产 | 杠杆+高周转 | 赚的钱全部要拿去买地，自由现金流常年为负 |
| 重资产制造 | 净利率+周转率不错 | 赚的钱要建厂买设备（资本开支大），FCF很少 |

### 判断步骤

看一家高ROE公司是不是现金奶牛，三步：

```
1. 杜邦拆解：ROE靠什么驱动？
   → 靠高利润率 → 可能是
   → 靠高杠杆 → 不是

2. 看自由现金流：
   → FCF > 净利润 × 50% → 是
   → FCF为负或极低 → 不是

3. 看分红/回购：
   → 大量分红或回购 → 是
   → 持续融资 → 不是
```

> 一句话：**现金奶牛 = 高ROE + 低杠杆 + 低资本开支 + 高自由现金流。** 只看ROE一个数字会骗人。
