## 1. 高ROE的公司都是现金奶牛吗

**不一定。** ROE高有三个不同的来源，只有一种是现金奶牛。

```
ROE = 净利润率 × 资产周转率 × 权益乘数
```

### 真正的高ROE现金奶牛

靠**极高的净利润率**驱动，杠杆很低，自由现金流丰厚。

```
茅台 ROE=30%：
  净利润率 49%  ← 核心驱动力
  周转率 0.5次
  权益乘数 1.23（几乎无杠杆）
  FCF/净利润 > 1（现金质量好，大量分红）
```

### 高ROE但不是现金奶牛的三种反例

| 类型 | 高ROE的原因 | 为什么不是现金奶牛 |
|------|-----------|----------------|
| 银行 | 杠杆极高（权益乘数>12） | 赚的钱必须保留为资本金应对监管，不能随便分，且FCF低 |
| 房地产 | 杠杆+高周转 | 赚的钱全部要拿去买地，自由现金流常年为负 |
| 重资产制造 | 净利率+周转率不错 | 赚的钱要建厂买设备（资本开支大），FCF很少 |

### 判断步骤

看一家高ROE公司是不是现金奶牛，三步：

```
1. 杜邦拆解：ROE靠什么驱动？
   → 靠高利润率 → 可能是
   → 靠高杠杆 → 不是

2. 看自由现金流：
   → FCF > 净利润 × 50% → 是
   → FCF为负或极低 → 不是

3. 看分红/回购：
   → 大量分红或回购 → 是
   → 持续融资 → 不是
```

> 一句话：**现金奶牛 = 高ROE + 低杠杆 + 低资本开支 + 高自由现金流。** 只看ROE一个数字会骗人。

## 2. ROE和增长率是一回事吗

**完全不是，一个是"效率"，一个是"速度"。**

| 概念 | ROE（净资产收益率） | 增长率（营收/利润增速） |
|------|-------------------|---------------------|
| 衡量什么 | 存量资本的赚钱效率 | 流量规模的扩张速度 |
| 公式 | 净利润 ÷ 净资产 | (本期 - 上期) ÷ 上期 |
| 单位 | 百分比（年化回报率） | 百分比（同比变化率） |
| 类比 | 车的**油耗效率**（百公里几升） | 车的**时速**（一小时跑多远） |
| 时间概念 | 截面的"比率"，分子分母都是当期 | 纵向的"变化"，两个时间点之差 |

**ROE的时间：** 净利润是一整年的累计数，净资产是这一年的平均值（或期末值）。
→ ROE 20% = 每$100净资产一年能赚$20，是**年化收益率**。

**增长率的时间：** 同样按年（YoY = Year-over-Year）。
→ 营收增长50% = 今年全年营收 ÷ 去年全年营收 - 1。

### 为什么20%的ROE看起来"不高"，但50%+的增长听起来"很高"

**因为它们衡量的是两个正交维度。**

```
      高增长 + 低ROE                 高增长 + 高ROE
      ┌─────────────────┐           ┌─────────────────┐
      │ 早期科技公司        │           │ NVIDIA (FY24-26)   │
      │ 营收翻倍但亏损      │           │ 增长126%→114%→65% │
      │ ROE为负            │           │ ROE > 50%         │
      └─────────────────┘           └─────────────────┘

      低增长 + 低ROE                 低增长 + 高ROE
      ┌─────────────────┐           ┌─────────────────┐
      │ 金辰股份（FY25）    │           │ 茅台               │
      │ 增长12%，ROE 2%    │           │ 增长~10-15%，ROE30%│
      │ 最差的一类          │           │ 现金奶牛           │
      └─────────────────┘           └─────────────────┘
```

**关键是：高增长率不可持续，高ROE可以持续。**

- 一家公司可以靠一个爆款产品突然营收翻倍，但竞争对手会跟进
- 高ROE（>20%）且持续多年的公司，说明有真正的护城河
- 50%+的营收增长通常持续不超过3-5年，而20%的ROE可以持续10年+

> 一句话：**ROE看你现有的钱赚得怎么样，增长率看你明年能赚多少。两者不是竞争关系，是互补关系。**

## 3. 50%以上营收增长的公司长什么样

增长率的计算口径——**同比（YoY）**：

```
营收增长率 = （今年全年营收 - 去年全年营收） ÷ 去年全年营收 × 100%
```

### 案例A：NVIDIA（NVDA）——AI算力的火箭式增长

| 财年 | 营收 | 同比增长 | 背景 |
|------|------|---------|------|
| FY2022 | $26.9B | +61% | 数据中心起步 |
| FY2023 | $27.0B | +0.2% | 游戏芯片崩盘 |
| **FY2024** | **$60.9B** | **+126%** | **AI大爆发，H100供不应求** |
| **FY2025** | **$130.5B** | **+114%** | **H100→H200+B200切换** |
| FY2026 | $215.9B | +65% | 基数太大，增速自然放缓 |

**50%+的增长是牛市级别的，但不可持续。** NVIDIA从$27B到$216B只用了3年，但这根曲线不可能永远这么陡。

### 案例B：宁德时代（300750）——高ROE+高增长结合的典范

| 年份 | 营收增长 | 净利润增长 | ROE |
|------|---------|-----------|-----|
| 2024 | +12% | +15% | 24.1% |
| 2025 | **+17%** | **+42%** | **24.9%** |

营收增长17%不算高，但利润增长42%（利润率在提升）+ 同时ROE 25%（资本效率高），是**高质量增长**。

### 案例C：药明康德（603259）——行业周期催生的爆发

2025年归母净利润同比增长**+103%**，海外biotech需求复苏+产能利用率提升驱动。翻倍的增长通常有特殊原因，不能线性外推。

### 案例D：比亚迪（002594）——高增长的终点

| 年份 | 营收增长 | 利润增长 |
|------|---------|---------|
| 2023 | +42% | +81% |
| 2024 | +29% | +26% |
| **2025** | **+3.5%** | **-19%** |

前几年新能源爆发时50%+的增长，进入到2025年后只有3.5%。**高增长总有尽头，关键看你在曲线的哪个位置。**

### 增长率的常见误区

```
误区1：把季度环比当成年化
  Q1增长20% → "年化80%？"
  不对。季度环比受季节性影响很大。看同比（YoY）才有意义。

误区2：利润增长50% = 公司变好了50%
  可能是卖资产、降一次性费用、或者去年基数极低。
  必须和营收增长匹配。营收+5%、利润+50% → 警惕。

误区3：高增长会永远持续
  NVIDIA从+126%降到+65%，比亚迪从+42%降到+3.5%。
  高增长无法长期维持是商业规律。

误区4：只看增速不看绝对值
  一家公司从$1M到$2M是+100%增长，从$100B到$150B是+50%增长。
  但后者一年增加的绝对值（$50B）是前者的50000倍。
  → 大公司增速慢是正常的。
```

### 总结

| 指标 | 核心含义 | 典型范围 | 可维持性 |
|------|---------|---------|---------|
| ROE | 资本赚钱效率 | 10-30% | 可以长期维持 |
| 营收增长率 | 规模扩张速度 | 0-50% | >30%很难持续超过3年 |
| 利润增长率 | 盈利提升速度 | -20%~100% | 波动极大，受一次性因素干扰 |

> 一句话：**ROE回答"你现在赚得怎么样"，增长率回答"你明年能赚更多吗"。一张表看质量，一张表看方向。**
