## 1. 滚动PE(TTM)、动态PE、静态PE到底是什么关系

**核心：中文软件里"动态PE"这个词被用滥了，不同平台指不同的东西。**

### 三个概念的定义

| 名称 | 公式分母 | 数据来源 | 性质 |
|------|---------|---------|------|
| **静态PE** | 最近一个完整财年的净利润 | 已披露年报 | 滞后但100%确定 |
| **滚动PE (TTM)** | 最近4个季度的净利润之和 | 已披露季报/年报 | 最新实际业绩 |
| **动态PE** | 预测的未来一年净利润 | 分析师预测 | 前瞻但靠猜 |

### 为什么软件上经常混用

**矛盾的根源：** 对比"静态PE"而言，TTM看起来更"动态"，所以很多中文平台直接把TTM标为"动态市盈率"。

| 平台 | "动态PE"实际指的是 | 备注 |
|------|-------------------|------|
| 东方财富 | 滚动PE (TTM) | 用最近4个季度实际利润 |
| 同花顺 | 滚动PE (TTM) | 同上 |
| 雪球 | 滚动PE (TTM) | 标为"PE-TTM" |
| 理杏仁 | TTM PE | 明确标注TTM |
| 部分港美股平台 | Forward PE | 用分析师预测 |

### 实战判断方法

拿到一个PE数字，先搞清楚分母用的是哪个利润：

```
股价 = 100元, EPS = 2元（上年年报） → 静态PE = 50
EPS = 2.5元（最近4个季度）       → 滚动PE = 40
EPS = 3元（分析师预测明年）      → 动态PE = 33
```

**同一个公司，同一个股价，三个PE分别是50、40、33——"便宜了还是贵了？"取决于你用哪个利润。**

### 自己算最靠谱

```
滚动PE(TTM) = 当前总市值 ÷ 最近4个季度净利润之和
```

不用依赖软件标注，手动算一下TTM就知道当前真实估值。

> 一句话：**静态PE看过去，TTM看现在，动态PE看未来。软件上标的"动态PE"大概率是TTM。**

## 2. 周期股为什么PE没法看——驰宏锌锗这类

**驰宏锌锗（600497）：** 有色金属（铅锌锗）矿企，标准周期股。

### 周期股的PE失真机制

```
矿企利润 = 矿产价格 × 产量 - 成本
          ↑ 完全由市场决定，公司无法控制
```

| 阶段 | 锌价 | 利润 | PE | 实际含义 |
|------|:----:|:----:|:--:|---------|
| 周期顶部 | 高 | 暴增 | **<10** | 利润已达顶峰，接下来要跌——看着最便宜时最危险 |
| 周期中部 | 回落 | 减少 | 20-30 | 正常估值 |
| **周期底部** | **低** | **微利/亏损** | **100+/无意义** | 行业出清中，反转可能临近——看着最贵时可能是机会 |
| 周期回升 | 上涨 | 恢复 | 15-25 | 利润在恢复，PE在回归 |

**驰宏锌锗PE=100+的原因不是公司变好了，而是锌价低迷导致利润降到极低。**

### 周期股的正确估值方法

```
不要看PE——用PB（市净率）
```

- 周期底部：PB接近或低于1 → 可能低估
- 周期顶部：PB > 2-3 → 可能高估

> 公式：**买周期股看PB低点，卖周期股看PB高点。PE在周期股上是反向指标。**

## 3. PE 0-10 都是什么情况——真实案例

**注意：PE = 0 不可能（分母净利润为0），但软件上"亏损"会显示为"—"或负数。**

A股常见的PE < 10 公司类型（2025年数据为参考）：

| 类型 | 典型公司 | PE | 为什么这么低 |
|------|---------|:--:|-------------|
| **强周期顶峰** | 中远海控 | ~3-5 | 海运价格历史高点，利润几百亿不可持续 |
| **低增长蓝筹** | 中国建筑 | ~4-5 | 营收万亿但增速<5%，市场不给溢价 |
| **行业至暗时刻** | 万科 | ~6-8 | 房地产行业信心崩塌，利润还可能继续降 |
| **银行** | 工商银行 | ~5-6 | 长期低增长+坏账隐忧，PB<1 |
| **煤炭顶峰** | 中国神华 | ~8-10 | 煤价高点利润大增，市场预期回落 |
| **被市场抛弃** | 金地集团 | ~3-4 | 房企，市场认为高负债有风险 |

### 低PE的两种本质

```
低PE = 真便宜 或 价值陷阱

真便宜：ROE>15%，利润持续稳定，现金流好
       但市场暂时不喜欢这个行业 → ESG/宏观情绪
       例：2022年的茅台PE=25（历史低位）

价值陷阱：ROE<10%，利润在缩水，负债高
         PE低是因为市场在"提前定价衰退"
         例：PE=5的房企，利润明年可能腰斩
```

> 一句话：**低PE要结合ROE和利润趋势来判断——ROE高+利润稳定+低PE是机会，ROE低+利润下滑+低PE是陷阱。**

## 4. 如何自己给一个估值区间（PE Band法）

不用DCF，用PE Band就能给一个实用的估值区间。

### 原理

合理价格 = 预测利润 × 合理PE

有四样东西要定：
1. 预测哪一年的利润（通常用今年或明年预测值）
2. 合理PE是多少
3. 乐观/悲观假设
4. 安全边际

### 实战：横店东磁

```
步骤①：确定利润基准
  FY2025 EPS = 1.16元
  2026E EPS ≈ 1.36元（多家券商一致预期）

步骤②：确定合理PE范围
  ROE = 17.42% → 参考合理PE = 15-20
  历史PE区间：12-25

步骤③：画估值区间
  悲观：1.36 × 12 = 16.3元  ← 历史底部+悲观情景
  中性：1.36 × 16 = 21.8元  ← 合理估值中枢
  乐观：1.36 × 20 = 27.2元  ← 高景气情景
  狂热：1.36 × 25 = 34.0元  ← 历史顶部

步骤④：对照当前股价判断
  如果股价在16-22元 → 合理区间，可以关注
  如果股价 < 16元   → 有安全边际
  如果股价 > 27元   → 偏贵，等回调
```

### 合理PE怎么定（参考表）

| ROE水平 | 增长情况 | 合理PE参考 | 例子 |
|---------|---------|:---------:|------|
| >20% | 高增长(>20%) | 25-35 | NVDA周期 |
| 15-20% | 稳定增长(10-20%) | 15-25 | 茅台、横店东磁 |
| 10-15% | 低增长(<10%) | 10-18 | 大部分A股蓝筹 |
| <10% | 持平/下滑 | <10 | 周期股底部/价值陷阱 |

### 注意事项

- **周期股不能用PE Band**（利润波动太大），改用PB Band
- 利润预测会不准，所以区间要留宽一些（±30%）
- **PE Band的本质假设：** 市场情绪会均值回归。历史PE区间不一定代表未来——如果公司基本面变了，合理PE的范围也会变

> 一句话：**估值区间 = 预测利润 × 合理PE。合理PE由ROE和成长性决定。区间不是精确值，是给你的"贵／便宜"参照系。**

## 5. PE在外围股票也通用吗

**通用。PE是全球通用的估值语言。** 美股、港股、欧股、日股都用PE。

### 不同市场的PE习性

| 市场 | 代表指数 | 典型PE范围 | 为什么 |
|------|---------|:----------:|--------|
| A股 | 沪深300 | 10-18 | 以金融+制造业为主，增速适中 |
| 美股 | S&P 500 | 18-25 | 科技+消费占比高，ROE更高 |
| 港股 | 恒生指数 | 8-15 | 金融+地产权重高，长期折价 |
| 日股 | 日经225 | 13-18 | 低增长+低利率环境 |
| 新兴市场 | MSCI EM | 10-16 | 高风险折价 |

**核心差异不是PE这个公式本身，而是不同市场的"基准PE"不同。**

### 跨市场对比PE时要小心

**错误用法：**

```
茅台PE=30 → 比可口可乐PE=25贵
→ 不对，A股消费股中枢PE本来就比美股高
```

**正确用法：同市场对比，或者同行业跨市场对比（但要调整市场溢价差异）。**

### 美股的特殊情况

| 情形 | 公司 | PE | 说明 |
|------|------|:--:|------|
| 高成长持续盈利 | 微软 | ~30-35 | ROE>30%，市场给溢价 |
| 亏损但高期待 | 特斯拉(早期) | 亏损/PE无意义 | 用PS估值 |
| 盈利但极低PE | 伯克希尔 | ~8-10 | 持有大量现金，市场给折价 |
| 超级成长 | NVDA(FY2025) | ~40-50 | 利润翻倍式增长，PEG≈1 |

### 核心一致：PE的判断逻辑全球通用

无论A股还是美股，判断逻辑一样：

```
① PE比历史中位数高还是低？
② PE比同行高还是低？
③ PEG是否合理？

→ 三个维度全球通用
```

> 一句话：**PE在全球都好用，但不同市场的"正常PE"不一样。跨市场对比要用行业相对PE，不要用绝对值。**

## 6. 破净（PB&lt;1）是机会还是陷阱

**核心答案：破净通常不是机会信号，而是警告信号。**

```
市场是有效的。一家公司的资产如果真的值那个价，它就不会破净。

破净 = 市场在说："你账上的资产，我不信值那么多钱。"
```

### 破净的三种本质

| 类型 | 例子 | 为什么破净 | 机会还是陷阱 |
|------|------|-----------|:-----------:|
| **资产虚胖型** | 银行（工行PB=0.6） | 坏账没有完全体现在净资产里，市场提前折价 | 陷阱，除非真实资产质量被严重低估 |
| **负债压顶型** | 房企（万科PB=0.4） | 净资产 = 总资产 - 总负债。高杠杆下，资产缩水10%净资产可能归零 | 陷阱，高杠杆下净资产脆弱 |
| **真正低估型** | 周期底部矿业 | 矿山/设备实际可产出价值高于账面价值 | **极少数**机会，需严谨验证 |

### 判断破净是否真低估——三步法

```
第一步：资产负债率 > 70%？
  → 高杠杆，净资产极脆弱，警惕

第二步：CFO是否为正且能覆盖利息？
  → 不赚钱的资产只会慢慢缩水

第三步：ROE是否 > 10%？
  → 如果资产真的值钱，为什么资本回报率这么低？
```

### 案例对比

```
工商银行   PB=0.6  ROE=10%  资产负债率=92%
  → 破净是不良贷款风险折价
  → 但ROE还行（>10%），现金流好
  → 不便宜也不贵，银行常态

某某房企   PB=0.3  ROE=2%   CFO为负
  → 破净是因为资产随时减值30%+
  → 利润和现金都在恶化
  → 典型的"价值陷阱"
```

### 总结

```
破净本身不意味着什么——它只是市场在说"有问题"。
问题是真问题还是假问题，要看：
  ① 负债高不高（资产负债率）
  ② 资产赚不赚钱（ROE）
  ③ 现金往哪流（CFO）
```

> 一句话：**破净 ≠ 便宜，破净 = 市场在质疑资产质量。是不是真便宜，靠L1/L2框架验证——ROE、负债率、CFO。**

## 7. PS（市销率）有数量标准吗

### PS没有固定的"低中高"标准

不像PE（0-10低/10-20中/20-30高）或PB（<1破净/1-3正常/3-5高），**PS没有统一标准，因为它完全取决于净利率。**

### 核心公式：PS = PE × 净利率

```
PS = 总市值 ÷ 营收
PE = 总市值 ÷ 净利润
→ PS = PE × (净利润÷营收) = PE × 净利率
```

**同一个公司，净利率越高，合理PS就应该越高。**

### 实用数值参考

| 净利率水平 | PE=15时 | PE=20时 | PE=30时 | 典型行业 |
|:---------:|:-------:|:-------:|:-------:|---------|
| 1%（薄利制造） | 0.15 | 0.20 | 0.30 | 化工、零售、代工 |
| 5%（一般制造） | 0.75 | 1.00 | 1.50 | 多数制造业 |
| 10%（消费品牌） | 1.50 | 2.00 | 3.00 | 家电、食品 |
| 20%（强品牌/软件） | 3.00 | 4.00 | 6.00 | 白酒、SaaS |
| 50%（极致盈利） | 7.50 | 10.00 | 15.00 | 茅台、部分垄断 |

### 快速判断规则

| PS范围 | 含义 | 例子 |
|:------:|------|------|
| **< 1** | 通常是低净利率生意，也可能被低估 | 大多数制造业、零售 |
| **1-3** | 普通消费品/制造业的合理范围 | 海尔、美的 |
| **3-5** | 较高利润率或品牌溢价 | 大部分消费品牌 |
| **5-10** | 高利润率（白酒/软件）或高增长预期 | 山西汾酒、Adobe |
| **> 10** | 极度乐观或亏损但高增长 | 早期SaaS、创新药 |

### PS的主要用途：弥补PE的死角

```
净利润为负时，PE无意义。

亏损的科技公司：营收10亿，市值100亿 → PS=10
→ 市场愿意为每1元营收付10元
→ 因为预期它未来会盈利

同样PS=10的亏损公司，情况可以完全不同：
  公司A：PS=10，毛利率80%，营收增长100% → 合理高PS（SaaS）
  公司B：PS=10，毛利率10%，营收增长5%  → 不合理（零售不该有高PS）
```

### 一个PS的常见误区

```
"PS=10比PS=1贵了10倍"→ 不一定。

公司A：PS=10，净利率20% → PE = PS÷净利率 = 50x
公司B：PS=1， 净利率5%  → PE = PS÷净利率 = 20x

→ 公司A看着PS贵，但PE 50x对高毛利高增长可能合理
→ 公司B看着PS便宜，但PE 20x对低毛利生意也正常
→ 不能只看PS绝对值
```

> 一句话：**PS没有绝对标准，PS = PE × 净利率。先想清楚净利率，PS才能有意义。PS最大的用途是给亏损公司一个估值参照。**

## 8. PCF（市现率）怎么用

**核心答案：PCF不单独使用，它是PE的"验钞机"。**

```
PE = 市值 ÷ 净利润    （利润口径，可修饰）
PCF = 市值 ÷ CFO     （现金口径，难造假）

PCF的作用不是告诉你"贵不贵"，而是告诉你"利润真不真"。
```

### 核心用法：PE vs PCF 对比

```
对比 PE 和 PCF，判断利润含金量：

情况A：PE < PCF
  → 净利润 > CFO，利润含金量不足
  → 应收高、存货积压、或提前确认收入
  → 例：某些房企PE=5但PCF=30+（利润在账上，钱没收到）

情况B：PE ≈ PCF
  → 净利润 ≈ CFO，利润真实
  → 正常

情况C：PE > PCF
  → CFO > 净利润，利润含金量高
  → 有预收款、折旧摊销大但没实际现金流出
  → 例：茅台PE=30, PCF=25（预收款多）
```

### 实战案例

```
横店东磁：PE ≈ 17, PCF ≈ 9.6
  → PE > PCF，CFO是净利润的1.76倍
  → 现金含量好 ✅

金发科技：PE ≈ 15, PCF ≈ 2.9
  → PE >> PCF，CFO是净利润的5.13倍
  → 现金含量极好 ✅（但利润基数低）

皖维高新：PE ≈ 39（TTM）, PCF ≈ 50+
  → PE < PCF，净利润 > CFO
  → 利润含金量不足 ❌
```

### 绝对值参考

| PCF | 含义 |
|:---:|------|
| < 10 | 偏低，可能低估或CFO高于利润 |
| 10-20 | 正常范围 |
| 20-30 | 偏高，CFO偏低 |
| > 30 | 警惕，利润含金量明显不足 |

### 一句话

> **PCF是PE的副手，不单独使用。用它只有一个目的——验证PE告诉你的"利润"到底有多少变成了真金白银。**

## 9. PEG是不是只对成长股有用

**是。PEG只适用于增长率 > 10-15%的成长股。成熟期低增长公司用PEG会严重误导。**

### 公式回顾

```
PEG = PE ÷ 净利润增长率（%）

判定标准：
  PEG < 1 → 可能低估
  PEG = 1 → 合理
  PEG > 1 → 可能高估
```

### 问题：为什么成熟公司增速低但PE不低？

**因为PE不仅是"增长"的函数，也是"确定性"和"现金流"的函数。** 成熟公司虽然增速低，但盈利确定、现金流好、分红稳定，市场愿意给一个"稳定性溢价"。

### PEG有用的"甜区" vs "禁区"

| 类型 | 例子 | 增速 | PE | PEG | PEG有效吗 | 原因 |
|------|------|:---:|:--:|:---:|:---------:|------|
| **高成长** | NVDA(FY2025) | +114% | 40 | 0.35 | ✅ 有效 | 高增长，PEG验证了PE合理性 |
| **稳定成长** | 横店东磁 | +15% | 17 | 1.1 | ✅ 基本有效 | 增速适中，PEG有参考价值 |
| **成熟低增长** | 茅台 | +12% | 30 | 2.5 | ❌ 无效 | 确定性溢价，PEG高估但不说明"贵" |
| **零增长** | 中国神华 | +3% | 10 | 3.3 | ❌ 完全无效 | 分红支撑估值，和增长无关 |
| **亏损** | 早期特斯拉 | 亏损 | N/A | N/A | ❌ 无意义 | PE为负，无法算PEG |

### 实战判断

```
用PEG之前，先回答一个问题：这家公司的增长可持续吗？

如果是：高速成长（>15%）且可持续  → PEG有用
如果是：短期的周期波动增长      → PEG无效（用PB）
如果是：已经到天花板了        → PEG无效（用PE+股息率）
```

### 什么时候该看什么

| 公司状态 | 核心估值指标 | PEG有用吗 |
|---------|:-----------:|:---------:|
| 高成长（增速>15%） | PE + PEG + PS（亏损时） | ✅ |
| 稳定成长（5-15%） | PE + PCF | ⚠ 参考，不主要 |
| 成熟低增长（0-5%） | PE + 股息率 + PB | ❌ |
| 周期股 | PB（低点判断） | ❌ |
| 亏损 | PS | ❌ |

> 一句话：**PEG只回答一个问题——"高PE是不是有高增长撑着"。对增速<10%的公司，PEG说了不算，看PE+股息率+PCF更实在。**

## 10. 股息率和分红率哪个更关键

**两个都要看，但说的不是同一件事。**

| 指标 | 公式 | 说的是 | 容易被什么干扰 |
|------|------|--------|-------------|
| **股息率** | 每股分红 ÷ 股价 | 买入后每年能拿回多少现金 | 股价跌→股息率被动升高 |
| **分红率** | 分红 ÷ 净利润 | 公司愿把多少利润分给你 | 利润波动→分红率忽高忽低 |

### 核心关系

```
股息率 = 分红率 ÷ PE

PE=5，  分红率50% → 股息率10%  ← 高股息来自低PE（可能周期谷底）
PE=30， 分红率90% → 股息率3%   ← 分红率高但股价贵，实际回报不高
```

### 实战场景

```
场景A：股价腰斩，股息率从3%被动升到6%
  → 高股息率是"假"的，因为利润可能也在跌
  → 先看分红率是否稳定，再看利润是否同步下滑

场景B：利润稳定，分红率从30%提高到80%
  → 股息率会涨
  → 但更要问：为什么突然多分钱？
      → 没有增长机会了？→ 成熟期正常
      → 借钱分红？→ 危险信号
```

### 案例分析

| 公司 | 分红率 | 股息率 | PE | 解读 |
|------|:-----:|:------:|:--:|------|
| 横店东磁 | 55-60% | ~2.5% | ~17 | 分红率慷慨，但PE不低所以股息率不高 |
| Suncorp | ~70% | ~4.5% | ~15 | 高分红率+中等PE→高股息，保险公司资本管理 |
| CBA | ~80% | ~4% | ~20 | 超高分红率，银行不需要大规模再投资 |
| 皖维高新 | 47.68% | ~2% | ~38 | 分红率适中，但高PE压低了股息率 |
| 三友化工 | ~50% | ~1.7% | ~175 | 超高PE（利润极低），高分红率但股息率极低 |

### 判断优先级的实用规则

```
① 先看分红率——判断"公司愿不愿意分"
   >70%：慷慨或成熟期，但注意可持续性
   30-70%：正常，留一部分再投资
   <30%：成长公司留利润扩张，或利润本身有问题

② 再看股息率——判断"现在买入划不划算"
   >5%：高股息，但警惕是不是股价跌出来的陷阱
   3-5%：良好，常见于成熟稳定公司
   <3%：普通或成长型公司
   0%：不分红，利润全部再投资

③ 最后做交叉验证——分红÷CFO是否<1
   CFO = 经营现金流
   如果分红 > CFO → 公司在借钱分红，不可持续
```

> 一句话：**分红率看"意愿"，股息率看"回报"。但高股息率可能是股价跌出来的陷阱，分红率突然飙升也可能有隐情。两者结合看，再配合分红÷CFO < 1 才安全。**

### 关键认知：股息率是防御指标，和成长无关

**股息率/分红率是收息型投资者的工具。如果你在找成长股，股息率越低反而越好。**

```
两种完全相反的投资逻辑：

成长型：利润 → 再投资 → 扩张 → 更高利润 → 股价涨
  → 分红是"浪费"，说明公司找不到好项目了
  → 所以高成长公司通常不分红或低分红

收息型：利润 → 分红 → 投资者拿现金
  → 分红是"回报"，追求稳定现金流
  → 所以高股息公司通常低增长甚至零增长
```

**预期回报拆解：**

```
股息率5%  + 利润增长0%  ≈ 年回报5%（全部来自分红）
股息率2%  + 利润增长15% ≈ 年回报17%（大部分来自股价涨）
股息率5%  + 利润增长15% → 不可能，增长型公司不会这么高分红
```

**所以看股息率之前，先问自己：**

```
你是在找成长股（看PE/PEG/PB）？
还是收息股（看股息率/分红率）？

→ 如果是前者，股息率 = 0% 完全没问题
→ 如果是后者，股息率 > 4% 是基本要求
```

## 11. 买高股息就能躺赚吗——总回报公式

**不能。买高股息≠睡得好。**

### 总回报公式

```
总回报 = 股价涨跌 + 股息收入
```

| 情景 | 股息率 | 股价涨跌 | 总回报 | 能睡好？ |
|:----:|:-----:|:--------:|:-----:|:--------:|
| CBA跌15% | 4% | -15% | **-11%** | ❌ |
| 横店东磁涨20% | 2.5% | +20% | **+22.5%** | ✅ |
| 雷曼兄弟（已破产） | 6% | -100% | **-94%** | ❌归零 |
| 稳定收息理想状态 | 4% | +5% | **+9%** | ✅ |

**股价跌30%，5%股息率根本扛不住。**

### "收息能睡好"的三个条件

高股息率本身不能让你睡好——**公司质量才是安全垫。**

| 条件 | 含义 | 反例 |
|------|------|------|
| **① 公司质量好** | ROE>15%，利润稳定，CFO健康 | 三友化工PE=175，利润跌82%，股息率再高也没用 |
| **② 股息可持续** | 分红率<70%，分红÷CFO<1 | 某些房企分红率>100%，借钱分红，随时砍息 |
| **③ 买入价格合理** | 在PE合理时买入 | CBA在PE=25时买，股息率被压低到3%，还可能承受估值回归 |

### 正确做法

```
❌ 错误：看股息率 → 闭眼买 → 睡大觉

✅ 正确：
  ① 找到ROE>15%、利润稳定的好公司
  ② 确认分红率合理（30-70%）、现金流能覆盖
  ③ 在合理PE时买入（不看股息率，看PE Band）
  ④ 长期持有，股息+利润增长共同推高回报
  ⑤ 接受股价有波动——只要公司没变坏，股息会持续发
```

> 一句话：**股息不是安全垫，公司质量才是。烂公司给你10%股息率你照样睡不好，好公司给你2%股息率但利润年增15%你反而能睡好——因为股价在涨。**

## 12. 企业价值（EV）是什么，为什么需要它

### PE的盲区

```
PE = 市值 ÷ 净利润
→ PE只看"股权"部分，忽略了公司背了多少债、手上有多少现金
```

**实际买一家公司时，你不仅要买它的股权，还要承担它的负债（但能拿走它的现金）。**

### EV公式

```
EV = 总市值 + 总负债 - 现金及等价物
```

简单理解：

```
买公司要花的真实总成本 = 买股票花的钱 + 替它还的债 - 它账上你拿到的现金
```

### 为什么需要EV——案例对比

```
公司A和公司B，利润都是10亿，市值都是200亿 → PE=20，看起来一样

但：
              公司A（轻资产）    公司B（重资产/高负债）
  市值              200亿             200亿
  总负债             20亿             200亿
  现金              100亿              20亿
  ─────────────────────────────────────────
  EV = 200+20-100    120亿          
  EV = 200+200-20                     380亿

  买A的真实成本只有120亿，买B要380亿
  → PE一样都是20，但真实投入差3倍
```

### EV/EBITDA——重资产行业的标准估值

```
EV/EBITDA = EV ÷ 息税折旧摊销前利润
```

为什么用EBITDA而不是净利润？因为EBITDA在扣利息和折旧之前，不受公司资本结构（借了多少钱、折旧年限怎么定）的影响，更适合跨公司比较。

| 范围 | 含义 |
|:---:|------|
| < 8 | 偏低 |
| 8-15 | 正常 |
| > 15 | 偏高 |

### 什么时候必须看EV

| 行业 | PE够用吗 | 原因 |
|:----:|:-------:|------|
| 轻资产消费（茅台） | ✅ 基本够 | 负债低，EV≈市值 |
| 普通制造业（横店东磁） | ⚠ 参考 | 负债中等，EV和市值有差别但不大 |
| **重资产高负债（金发科技/电力/钢铁）** | **❌ 不够** | **负债远大于现金，PE严重低估真实成本** |
| 银行/保险 | ❌ 不用 | 负债性质不同（存款是负债但不是"债"），看PB |

### 金发科技实战

```
金发科技：
  市值 ≈ 180亿
  短期借款45亿 + 长期借款159亿 + 其他 ≈ 总负债233亿
  现金43亿

  EV = 180 + 233 - 43 = 370亿

  PE = 180 ÷ 11.5 = 15.6x  ← 看着"合理"
  EV/EBITDA = 370 ÷ 约22（折算） ≈ 16.8x  ← 中性偏贵

  → PE说"还行"，EV说"兄弟它欠了两百多亿"
  → 金发科技欠的钱确实多——PE好看的背后是EV在报警
```

### 一句话

> **EV把PE漏掉的负债补回来了。轻资产公司PE够用，重资产高负债公司必须看EV/EBITDA。否则就会像金发科技一样——PE15倍看着便宜，EV告诉你真实成本要翻倍。**
