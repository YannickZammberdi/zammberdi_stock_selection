# Stock Learning - 股票投资学习项目

## 学习模式

本项目的学习采用 **"先读资料 → 后做Quiz → 记录进度"** 的循环模式。

| 关卡 | 主题 | 状态 |
|------|------|------|
| L1 | 财务三表入门 | ✅ 已通过 |
| L2 | 关键财务指标 | ✅ 资料就绪 |
| L3 | 估值指标 | ✅ 资料就绪 |
| L4 | 杜邦分析拆解 | ✅ 资料就绪 |
| L5 | 行业分类与特征 | ✅ 资料就绪 |
| L6 | 综合实战 | ✅ 资料就绪 |

## 个股实战模式

你给我股票代码/名称 → 我在 `stocks-analysis/` 中建立分析档案 → 你阅读分析档案后提交自己的判断 → 我评分对比。

## 目录结构

```
stock-learning/
├── README.md              ← 本文件
├── progress.md            ← 学习进度记录
├── materials/             ← 学习资料（先读后练）
│   ├── L1-financial-statements.md
│   ├── L2-key-metrics.md
│   ├── L3-valuation.md
│   ├── L4-duPont.md
│   ├── L5-industry.md
│   └── L6-comprehensive.md
├── notes/                 ← 展开笔记（每L配一个N，含常见追问）
│   ├── N1-how-to-judge-high-low.md
│   ├── N2-xxx.md
│   ├── N3-xxx.md
│   ├── N4-xxx.md
│   ├── N5-xxx.md
│   └── N6-xxx.md
└── stocks-analysis/       ← 个股分析档案
    ├── template.md        ← 分析模板
    └── index.md           ← 已分析股票清单
```

## 笔记说明

每章 Lx 末尾有链接指向对应的 Nx 笔记文件。N 文件收录的是学习过程中可能产生的追问和扩展知识，比如"XX指标怎么看高还是低"、"XX和XX有什么区别"。你在学习过程中想到的问题也会被整理进去。
