 • 控股股东增持+分红率40%，利益一致性强

 • 当前非周期底部（已在中周期），配额红利已部分被股价反映

行业分类与特征（L5框架）

行业类型判断

行业类型：纯周期股（基础化工-氟化工）
判断依据：巨化股份的核心产品氟制冷剂是典型的周期品——价格受供需错配驱动大幅波动，利润弹性极大（FY2023净利9.4亿→FY2025净利37.8亿，3年翻4倍）。虽然配额制（基加利修正案）从供给端锁定了部分产能，减少了周期波幅，但行业本质上仍属于强周期。

选用的评分体系：周期股评分体系
评分理由：纯周期企业必须用PB估值、关注穿越周期ROE均值、重视周期位置判断。如果用非周期股评分体系（高PE扣分多），反而会误判——周期股盈利高峰时PE最低，用PE估值会高估安全边际。

行业集中度与壁垒

维度判断
行业集中度高集中度（配额制锁定）——巨化股份三代制冷剂配额占全国~39%，CR3~60%。不同于普通化工品，配额是不可逾越的供给天花板。
政策壁垒极高——基加利修正案（2016）冻结HFCs产量并计划逐步削减。2024年起中国执行生产配额制，新进入者无法获得配额。这是化工行业最强的准入壁垒之一。
产业链壁垒高——从萤石→氢氟酸→制冷剂/聚合物，全产业链一体化需要数十亿投资和多年积累。
技术壁垒中等——制冷剂生产技术已成熟，但高附加值产品（如电子级氟化液）仍有壁垒。

生命周期阶段

阶段：成熟期（但配额制改变了生命周期曲线）
制冷剂行业本身已进入成熟期（增速<5%），但2024年配额制实施本质上是"供给侧改革"——通过行政手段限制产量，将行业从"充分竞争的周期品"转变为"配额管制的特许经营品"。这在一定程度上延长了行业生命周期并抬高了利润中枢。

投资策略重点：关注制冷剂价格走势、2027-2029年配额削减进度、甘肃项目投产节奏、含氟精细化学品放量。

当前周期位置

周期位置判断：中周期（复苏→繁荣过渡期）
制冷剂价格自2024年配额制启动后持续上涨，当前处于价格上行中段。本轮周期的特殊性在于——供给端有配额制做刚性约束，不像传统化工周期那样纯粹靠市场出清。因此本轮高景气持续时间可能比传统周期更长（预计2-3年），但价格不可能无限上涨。

对评分的影响：周期位置评分7/15——既非底部区间（+15分）也非暴利期（+2分），属于中周期阶段。当前买入并非"逆周期布局"，而是"顺周期持有"阶段。

 乐观：R32维持3.5-4.5万/吨 → EPS 2.0-2.3元
 中性：R32在3.0-3.5万/吨区间波动 → EPS 1.6-1.9元（当前基准）
 悲观：下游需求走弱+价格回落至2.5万以下 → EPS 1.0-1.3元

变量2：甘肃巨化项目投产回报
 乐观：2026Q3顺利投产+产品高附加值 → 2027年贡献利润5-8亿
 中性：按期投产但爬坡慢 → 2027年贡献利润2-3亿
 悲观：延期或产品价格低迷 → 拖累整体ROE 2-3pct

变量3：配额政策变动（2027-2029削减节奏）
 乐观：削减进度延缓或维持现有配额 → 景气周期延长至2028+
 中性：按既定计划逐步削减5-10%/年 → 供给趋紧支撑价格
 悲观：提前加速削减或新增豁免条款 → 不确定性增加
